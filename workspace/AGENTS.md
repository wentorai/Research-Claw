# Research-Claw Workspace

This is a **Research-Claw** workspace. Agent behavior specification is defined in
`.ResearchClaw/AGENTS.md` — that file is the authoritative source for session
startup, tool priority, card formatting, HiL protocol, and all operational
procedures.

Do NOT follow generic OpenClaw workspace conventions here. Research-Claw has its
own module system (Library, Tasks, Workspace, Monitor, Radar) and its own
bootstrap file layering (L1/L2/L3). See `.ResearchClaw/` for all details.

## Quick Reference

- **Full agent spec:** `.ResearchClaw/AGENTS.md`
- **Tool catalog:** `.ResearchClaw/TOOLS.md`
- **Persona:** `.ResearchClaw/SOUL.md`
- **Identity:** `.ResearchClaw/IDENTITY.md`
- **Heartbeat routine:** `.ResearchClaw/HEARTBEAT.md`
- **User profile:** `.ResearchClaw/USER.md` (L3 — you own this)
- **Memory:** `MEMORY.md` (L3 — you own this)
