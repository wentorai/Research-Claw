# 中文期刊配图规范全集

面向 CSSCI / 北大核心 / C 刊的配图与表格规范。涵盖《经济研究》《管理世界》《会计研究》《心理学报》《中国工业经济》《金融研究》《南开管理评论》《系统工程理论与实践》等。

## 1. 字体（Font）

### 1.1 中文字体优先级（fallback chain）

期刊投稿与排版用宋体；正文 / 图注 / 表注 / 标题统一用宋体。

```python
plt.rcParams['font.serif'] = [
    'Times New Roman',      # 西文与数字
    'SimSun',               # Windows 宋体
    'Songti SC',            # macOS 宋体
    'Source Han Serif SC',  # 思源宋体（Linux 推荐）
    'STSong',               # 备选华文宋体
    'Noto Serif CJK SC',    # 备选 Noto
]
plt.rcParams['axes.unicode_minus'] = False  # 必须，否则负号显示成方块
```

### 1.2 西文字体

数字与英文一律 **Times New Roman**。
不要用 Helvetica / Arial（那是英文期刊的偏好）。

### 1.3 字号（pt）

| 元素 | 推荐字号 | 说明 |
|------|---------|------|
| 图标题 | 10–11 | 论文正文一般 10.5pt（小四），图标题与之一致或小一号 |
| 坐标轴标签 | 9–10 | 例如 "处理效应" / "年份" |
| 刻度标签 | 8–9 | 数字 |
| 图例 | 8–9 | |
| 图注（注：……） | 8 | 五号或小五号 |

注意：matplotlib 默认 10pt 在缩到单栏 8.5cm 后会偏小，推荐**绘图时设为 10pt，导出时不要再缩放**（使用最终尺寸 figsize）。

## 2. 图幅尺寸

| 版式 | 最大宽度 | 推荐 figsize (英寸) |
|------|---------|--------------------|
| 单栏（多数 C 刊） | 8.5 cm ≈ 3.35 in | (3.3, 2.5) |
| 双栏 / 整版心 | 17 cm ≈ 6.7 in | (6.5, 4.0) |
| 整版心 + 较高 | 17 × 12 cm | (6.5, 4.7) |

**总宽度不超过版心的 80%**——给图注与白边留余地。

宽高比建议：
- 折线 / 时间序列：**4:3** 或 **golden ratio 1.618:1**
- 系数图（forest plot）：**4:5** 或 **3:4**（窄而高）
- 热图 / 相关矩阵：**1:1**

## 3. 颜色

### 3.1 黑白可读优先

中文 C 刊大量印刷仍是黑白；图必须**在不上色的情况下也可读**。即：

- **同时**用 marker (`o, s, ^, D, v, x`) 与 linestyle (`-, --, -., :`) 区分序列
- 颜色只是**冗余信息**，不能是唯一区分手段

### 3.2 推荐调色板

不要用 matplotlib 默认 `tab10`（黑白打印偏色严重）。推荐：

**方案 A（黑白 + 灰阶）**——首选投 C 刊
```python
COLORS_BW = ['#000000', '#666666', '#999999', '#CCCCCC']
LINESTYLES = ['-', '--', '-.', ':']
MARKERS = ['o', 's', '^', 'D']
```

**方案 B（色盲友好 Okabe-Ito 8 色）**——彩印 / 网络版
```python
COLORS_OI = [
    '#000000',  # black
    '#E69F00',  # orange
    '#56B4E9',  # sky blue
    '#009E73',  # bluish green
    '#F0E442',  # yellow
    '#0072B2',  # blue
    '#D55E00',  # vermilion
    '#CC79A7',  # reddish purple
]
```

**方案 C（Paul Tol 6 色定性）**——彩印 / 期刊封面候选
```python
COLORS_PT = ['#332288', '#117733', '#44AA99', '#88CCEE', '#DDCC77', '#CC6677']
```

详见 `colorblind-bw.md`。

## 4. 线宽与 marker 尺寸

| 元素 | 推荐值 | 说明 |
|------|-------|------|
| 主数据线 | 1.2–1.5 pt | 默认 1.5 pt 对 C 刊版心刚好 |
| 坐标轴 / 刻度线 | 0.6–0.8 pt | |
| 网格线（如开） | 0.4 pt + alpha=0.4 | 通常不开 |
| Marker 大小 | 4–6 pt | 时间序列 5pt，散点视密度 |
| 误差线 | 1.0 pt + capsize=3 | |

**禁忌**：默认 0.5 pt 的细线在缩印后会消失。

## 5. 坐标轴

- **保留 4 边框**（matplotlib 默认）或**只保留左 + 下**（去 spine top/right）。两种风格都可，C 刊更常见后者
- **刻度向内**（`tick_params(direction='in')`），西文期刊常用向内向外，中文期刊偏好向内
- **不开网格** —— 默认不开。若数据需要参考线，开横向虚线 + alpha=0.3
- **数字 + 单位**：纵轴标签写"利润率（%）"或"GDP（亿元）"，单位放圆括号

## 6. 图例

- **位置**：图内右上 / 左上，避免压住数据；点多时放图外
- **frameon=False**（不画框）或 frameon=True + 浅灰边框 0.5pt，二选一
- **fontsize**：8–9 pt
- **不要写英文 Legend** —— 中文期刊全中文："实验组 / 对照组"，不要 "Treated / Control"

## 7. 图注（Caption）

- **位置**：图下方
- **格式**：`图1 ××××的影响 \n 注：……`
- 第一行是图标题（图序 + 标题），第二行起是图注本身
- **图序**：中文"图1"、"图2"，**不是** "Figure 1"
- 图注内容：
  1. 数据范围 / 样本（"样本：2010—2022 年沪深 A 股，N = 38,420"）
  2. 关键术语解释（变量定义如必要）
  3. 数据来源（"数据来源：CSMAR 数据库。"）
  4. 显著性符号说明（如有）

## 8. 多子图（subplots）

- 子图标号：**(a) (b) (c)** 小写括号字母，置于子图**左上角**或**子图标题前**
- **不要**用 "1." "2." 数字
- 子图之间留 **wspace=0.3, hspace=0.4** 左右
- 共享坐标轴时（`sharex / sharey`），只在最外圈显示标签
- 整体标题用 `fig.suptitle`，但中文期刊一般**不写整体标题**，标题写在 caption 里

## 9. 导出

```python
fig.savefig('fig1.pdf', dpi=300, bbox_inches='tight', pad_inches=0.05)
```

- **格式**：PDF（矢量，首选）；EPS 也可（部分老期刊要求）
- **不要交 PNG / JPG** —— 排版后糊成一团
- **dpi=300** 是底线，dpi=600 更稳
- `bbox_inches='tight'` 自动裁掉白边，但保留 `pad_inches=0.05` 一点呼吸空间

## 10. 文件命名

- `fig1_main_did.pdf`、`fig2_event_study.pdf` 这种描述性名字
- **不要**用 `Untitled.pdf`、`图.pdf`、`新建.pdf`
- 编号与正文中"图1 / 图2"对应
