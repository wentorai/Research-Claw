# Journal Table Formats

Journal-specific instructions are authoritative. Use this file as a conservative starting point.
## 经济研究
- Chinese title, Chinese notes, three-line table, no vertical lines.
- Prefer compact tables that remain readable in print.
- Define sample, controls, fixed effects, and standard errors in notes.
## 管理世界
- Chinese labels are preferred for domestic submissions.
- Keep table notes complete: data source, variable definitions, clustering, fixed effects.
- Avoid excessive decimal places.
## Journal of Finance
- English title and notes, booktabs style, clear model columns.
- Report N, adjusted R-squared where appropriate, fixed effects, and clustered SE.
- Keep appendix tables consistent with main tables.
## Management Science
- Strong preference for transparent model comparison and reproducible notes.
- Make identification and model changes legible across columns.
- Do not overload the main manuscript with many near-identical robustness columns.
## Cross-journal common rules
- No vertical lines.
- Same variables in same row order across related tables.
- Coefficient and SE appear in the same column cell, coefficient above SE.
- FE and controls are summarized in rows at bottom.
- Include data/code availability notes when journal requires them.
