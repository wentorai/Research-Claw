# 实证结果写作提示词

> 实证结果章节是审稿人评判论文质量的重要部分。本提示词帮助 Claude 写出结构清晰、解释克制、与表格图形紧密配合的"实证结果"小节。

## 章节通用结构

1. **描述性统计**：1 张表，1-2 段文字。重点写"哪些变量分布特征值得注意"，不要逐行复读表内数字。
2. **基准回归**：1 张主表，2-4 段文字。先报告核心系数与显著性，再对比加入控制变量后的稳定性，再讨论经济意义（系数转换为标准差、百分点、弹性等更直观的口径）。
3. **稳健性**：分小节呈现，每个稳健性检验 1 段，写明"针对何种担忧—如何处理—结果是否稳健"。
4. **异质性**：按组别（地区、规模、行业、时间窗口等）展开，每组 1 段。
5. **机制分析**：每条机制 1-2 段，写明"理论预期—中介或交互项设计—结果是否支持"。

## 写作要点

- **不要复读系数**：表格里写得清楚的数字不必在正文中再重复一遍。正文要做的是"指方向、给口径、回应识别担忧"。
- **经济意义**：仅有"统计显著"是不够的，要把系数翻译成读者能直观理解的口径（如"约 0.X 个标准差""相当于均值的 X%"）。
- **与图表呼应**：每提到一张表都要写"如表 X 所示"或"见表 X 第（Y）列"。
- **时态与人称**：中文学术文本用陈述语气，避免"我们发现"，可改为"本文发现"或被动构造的"研究表明"。
- **不要把稳健性写成清单**：每条稳健性都要解释 why（针对哪个担忧）。

## 提示词模板

```
你将作为中文经济学顶级期刊的写作教练，为以下实证结果章节起草中文正文。

【基准回归方程】
Y_{i,t} = α + β · Treatment_{i,t} + γ · X_{i,t} + μ_i + λ_t + ε_{i,t}
（请根据实际研究替换为正确的方程）

【主要结果】
- 基准回归核心系数 β = {{数值}}，标准误 = {{数值}}，显著性 = {{星号水平}}
- 经济意义：{{把系数转换为更直观的口径，如"被解释变量上升约 X 个百分点"}}

【稳健性检验列表】
1. {{检验 1：针对何种担忧 + 检验方法 + 结果}}
2. {{检验 2：…}}
3. {{检验 3：…}}

【异质性分析】
{{按 X 维度分组，结果方向}}

【机制】
{{机制 1 名称 + 中介/交互项设计 + 结果}}
{{机制 2 名称 + 中介/交互项设计 + 结果}}

请输出实证结果章节正文，包含 5 个小节：
1. 描述性统计
2. 基准回归（包括结果解读与经济意义讨论）
3. 稳健性检验（每个检验 1 段）
4. 异质性分析
5. 机制检验

写作约束：
- 全文中文，学术语气
- 在合适位置插入"如表 X 所示""见图 X"等指示语
- 不要复读所有表内数字，只点出方向、显著性与经济意义
- 不要给出回归方程的全部下标解释（已在研究设计章节给出）
- 不要使用"非常""极其""极强"等程度副词
- 稳健性部分每条要写明"针对何种担忧"
- 异质性结果要给出"为什么会有此异质性"的简短解释
- 机制部分要回到引言中提到的理论预期
```

## 自检清单

- [ ] 是否在每张表/图首次出现时都加了"如表 X 所示"？
- [ ] 是否把核心系数解释为经济意义（标准差、百分点、弹性等）？
- [ ] 稳健性检验是否每条都有"针对何种担忧"的解释？
- [ ] 异质性是否给出"为什么"的解释？
- [ ] 机制部分是否与理论假说对应？
- [ ] 是否避免了"我们发现""非常显著""极其稳健"等用语？
- [ ] 段落之间是否有过渡，而非生硬罗列？
