# R&R 评审回复提示词

> 收到《经济研究》《金融研究》《管理世界》等期刊的修改意见后，回复信的质量直接决定二审结果。本提示词帮助 Claude 起草礼貌、严谨、可逐条对照的中文回复。

## 回复信的基本结构

1. **致编辑的总信**（1 页内）
   - 感谢编辑与匿名审稿人的细致评阅
   - 简述本轮主要修改内容（3-5 条提纲式概括）
   - 说明随附文件清单（修订稿、回复信、复现包等）

2. **逐条回复**（每位审稿人单独一份）
   - 审稿人原意见（**完整复制**，不要改写或缩短）
   - 作者回复（先表态，再说明修改方案，再指明在修订稿中的位置）
   - 修订稿对应位置摘录（带页码或行号）

## 回复语气准则

- **始终礼貌**：每条回复以"感谢审稿人指出 …"或"非常感谢审稿人的建议"开头
- **不与审稿人争辩**：即便不同意审稿人的判断，也要先肯定意见的合理性，再给出"补充证据"而非"反驳"
- **明确"做了什么"**：含糊的"我们已修改"是不够的，要写明"我们在第 X 节新增了 … 表 X / 图 X / 段落"
- **量化呈现新增工作**：例如"新增 3 项稳健性检验，详见 4.5 节"
- **必要时承认不足**：对于无法完全解决的问题，可写"由于数据限制，我们无法 …，但通过 … 进行了间接检验"

## 提示词模板

```
你将作为中文经济学顶级期刊的写作教练，为以下 R&R 起草回复内容。

【期刊与论文】
期刊：{{《经济研究》/《金融研究》/...}}
论文标题：{{中文标题}}
轮次：{{第一轮修改 / 第二轮修改}}

【审稿人意见原文】
{{完整复制审稿人意见，逐条用"意见 1""意见 2"…标记}}

【作者拟回应方向】
意见 1：{{接受 / 部分接受 / 解释；具体修改方案}}
意见 2：{{...}}
…

【修订稿主要变化】
- {{变化 1：在第 X 节新增/修改了什么}}
- {{变化 2：...}}

请按下面结构输出回复：
A. 致主编与审稿人的总信（不超过 1 页，约 400-600 字）
B. 逐条回复
   - 每条以"审稿人意见 X" + 完整原文开头
   - 然后是"作者回复"，分三部分：
     1. 致谢与态度
     2. 修改方案（具体到新增的表、图、段落、检验）
     3. "修订稿中相应位置：第 X 节，见正文/表 X"

写作约束：
- 全文中文，敬语规范（"审稿人""您"）
- 每条回复必须包含具体的修订位置（章节、表号、图号）
- 不与审稿人争辩；不同意见也以"我们补充了 …"的方式呈现
- 不使用"开创性""非常重要""强烈支持"
- 涉及新增的表/图，回复信中要简要描述其内容（1-2 句）
- 致谢部分避免重复套话，建议每位审稿人的致谢略有差异
```

## 处理"刁钻"意见的话术参考

- **审稿人怀疑识别**：先说"感谢审稿人对识别的关切，这是一个非常重要的问题"，然后给出"我们在 X 节新增了 [安慰剂/平行趋势/弱工具检验/边界检验]，结果显示 …"。
- **审稿人要求加大量额外数据**：若数据可获得，承诺补充；若不可获得，说明限制并给出"我们以 … 进行了替代性检验"。
- **审稿人质疑机制**：先肯定"机制确实是审稿人指出的核心问题"，然后给出新增的中介/交互检验或异质性证据。
- **审稿人质疑外部效度**：承认局限，并指出已在结论中限定结论适用范围。
- **审稿人要求改变叙事**：对于纯写作建议，可全部接受并说明"已在 … 修改"。
- **审稿人误读了原文**：礼貌地说"由于此处表述不够清晰，可能造成误解。我们在第 X 节重新表述为 …"，**不要**说"审稿人误解了"。

## 自检清单

- [ ] 是否每条意见都有独立编号且原文完整复制？
- [ ] 每条回复是否给出了具体修订位置？
- [ ] 是否避免了与审稿人争辩的语气？
- [ ] 致谢是否避免了套话堆砌？
- [ ] 是否在总信开头简述了"主要修改"？
- [ ] 修订稿摘录的页码/行号是否与最终版本对得上？
- [ ] 文件命名是否清晰（如 `Response_to_Reviewers_R1.pdf`）？
