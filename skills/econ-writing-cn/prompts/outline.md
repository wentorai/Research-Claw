# 论文提纲生成提示词

> 当用户给定一个研究问题与数据，让 Claude 生成中文经济学论文的章节提纲时，使用本提示词。

## 用法

复制下方"提示词模板"中的内容，替换 `{{...}}` 占位符后发给 Claude。

## 提示词模板

```
你将作为一名经验丰富的中文经济学论文写作教练，为以下研究生成完整的章节提纲。

【研究问题】
{{用一句话描述：本文要回答的核心因果/描述性问题}}

【数据】
{{数据来源、时间区间、样本规模、关键变量}}

【识别策略】
{{DID / IV / RDD / SC / Event Study / Bartik / 结构估计 / 描述性 等}}

【目标期刊】
{{《经济研究》/《金融研究》/《管理世界》/《会计研究》/《管理科学学报》/《数量经济技术经济研究》}}

请按下列结构输出：
1. 标题（中文，不超过 25 字；英文标题可选）
2. 中文摘要骨架（200-300 字，按"问题—方法—发现—含义"四要素，发现部分先留占位）
3. 关键词（3-6 个）
4. 章节提纲（一级、二级标题；每个二级标题下用 1-2 句话说明该节核心内容；不要写正文）
5. 主表与主图清单（建议 1 张主图、4-6 张主表）
6. 稳健性检验清单（至少 5 项，覆盖样本、变量、估计量、安慰剂）
7. 机制检验设计（2-3 条机制，每条说明被检验变量与预期方向）
8. 政策含义提纲（3 条具体的、可操作的建议）

要求：
- 全部使用中文
- 章节命名遵循目标期刊的常见风格
- 不要写正文段落，只输出提纲
- 不要使用"开创性""填补空白""非常重要"等用语
```

## 进阶选项

如需为同一研究生成两个不同侧重的提纲（例如同时投《经济研究》与《金融研究》），可在提示词后追加：

```
请额外输出针对 {{第二期刊}} 的提纲版本，重点突出 {{该期刊偏好的角度}}。
```

## 自检清单

收到 Claude 输出后，建议自查：

- [ ] 标题是否点明了研究对象与方法？
- [ ] 章节是否涵盖：引言、文献与理论、制度背景、研究设计、实证结果、稳健性、机制、异质性、结论与建议？
- [ ] 主表清单是否包含描述性统计、基准回归、机制、异质性？
- [ ] 主图是否至少包含一张能"一眼看出识别"的图？
- [ ] 稳健性是否涵盖样本、变量、估计量三类替换？
- [ ] 政策建议是否具体可操作（不是"加强监管"这类空话）？
