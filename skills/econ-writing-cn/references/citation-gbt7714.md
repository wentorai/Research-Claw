# GB/T 7714-2015 引用规范示例

GB/T 7714-2015《信息与文献 参考文献著录规则》是中国国家标准。下文按"顺序编码制"给出常见文献类型的著录示例。中文期刊大多采用此格式或在此基础上略作修改。

## 文献类型标识符

著录中常见的中括号类型标识符：

| 类型 | 标识符 |
| --- | --- |
| 期刊论文 | [J] |
| 专著 / 普通图书 | [M] |
| 论文集 / 会议录 | [C] |
| 会议论文（论文集中析出） | [C]/[A]（早期版本） |
| 学位论文 | [D] |
| 报告 | [R] |
| 标准 | [S] |
| 专利 | [P] |
| 报纸文章 | [N] |
| 电子文献 | [EB/OL]（在线电子文献）、[DB/OL]（在线数据库）等 |

电子文献可同时附加载体类型标识，如 `[J/OL]` 表示在线期刊论文。

---

## 1. 期刊论文 [J]

**格式**：

```
主要责任者. 题名[J]. 刊名, 出版年, 卷(期): 起止页码.
```

**中文示例**：

```
张三, 李四. 数字化转型对企业生产率的影响——基于上市公司的经验证据[J]. 经济研究, 2024, 59(3): 45-62.
```

**英文示例**：

```
Acemoglu D, Restrepo P. Robots and Jobs: Evidence from US Labor Markets[J]. Journal of Political Economy, 2020, 128(6): 2188-2244.
```

> 注意：英文作者姓在前，名取首字母，多个作者用逗号分隔；最后一个作者前不加 "and"。

---

## 2. 专著 / 普通图书 [M]

**格式**：

```
主要责任者. 题名: 其他题名信息[M]. 其他责任者. 版本项. 出版地: 出版者, 出版年: 引文页码.
```

**中文示例**：

```
林毅夫. 中国的奇迹: 发展战略与经济改革[M]. 增订版. 上海: 上海三联书店, 2014: 88-102.
```

**英文示例**：

```
Wooldridge J M. Econometric Analysis of Cross Section and Panel Data[M]. 2nd ed. Cambridge, MA: MIT Press, 2010: 245-260.
```

**译著示例**：

```
阿西莫格鲁 D, 罗宾逊 J A. 国家为什么会失败[M]. 李增刚, 译. 长沙: 湖南科学技术出版社, 2015.
```

---

## 3. 会议论文 [C]

**论文集整体作为参考文献**：

```
主要责任者. 题名[C]. 出版地: 出版者, 出版年.
```

**从论文集中析出的单篇论文（推荐格式）**：

```
析出文献主要责任者. 析出文献题名[C]//论文集主要责任者. 论文集题名. 出版地: 出版者, 出版年: 起止页码.
```

**中文示例**：

```
王五. 基于机器学习的信用风险预测[C]//中国数量经济学会. 中国数量经济学会 2023 年年会论文集. 北京: 中国数量经济学会, 2023: 121-138.
```

**英文示例**：

```
Smith J. Causal Inference with Synthetic Control[C]//Proceedings of the NBER Summer Institute. Cambridge, MA: NBER, 2022: 45-60.
```

---

## 4. 学位论文 [D]

**格式**：

```
主要责任者. 题名[D]. 保存地: 保存者, 年份.
```

**示例**：

```
赵六. 货币政策对企业投资的异质性影响研究[D]. 北京: 中国人民大学, 2022.
```

```
Doe J. Essays on Labor Economics[D]. Cambridge, MA: Harvard University, 2021.
```

---

## 5. 电子文献 [EB/OL] / [J/OL] / [DB/OL]

**格式（在线电子文献）**：

```
主要责任者. 题名[载体类型标识]. 出版地: 出版者, 出版年[引用日期]. 获取或访问路径. DOI（如有）.
```

**示例（在线期刊论文）**：

```
钱七. 绿色金融政策的环境治理效应[J/OL]. 金融研究, 2024, 50(2): 1-18[2026-04-15]. https://example.com/article/12345.
```

**示例（统计数据库）**：

```
国家统计局. 中国统计年鉴 2023[DB/OL]. 北京: 国家统计局, 2023[2026-04-20]. https://www.stats.gov.cn/.
```

**示例（在线工作论文 / 报告）**：

```
Chen X. Working Paper on Trade and Labor[R/OL]. NBER Working Paper No. 30000. Cambridge, MA: NBER, 2023[2026-04-30]. https://www.nber.org/papers/w30000.
```

## 排序与编号约定

- 顺序编码制：参考文献按文中首次引用的先后顺序排列、编号；正文中以 `[1]`、`[2,3]`、`[4-7]` 等形式标注。
- 著者-出版年制：正文中以 "(作者, 年份)" 标注，参考文献按作者姓氏拼音/字母顺序排列。**国内多数期刊采用顺序编码制**，但《管理世界》等部分期刊近年偏向著者-出版年制，请以期刊投稿须知为准。

## 常见错误

1. 中文作者只写姓不写名（除非作者本人确实只用单字名）。
2. 英文作者全名写出而未取首字母（如写 "Acemoglu Daron"）。
3. 期刊名缩写用错（中文期刊一般不缩写）。
4. 期、卷标注错位（中文期刊通常给出 `年, 卷(期)`）。
5. 电子文献缺少引用日期或访问路径。
6. 同一作者多篇文献排列时未注意年份顺序。
