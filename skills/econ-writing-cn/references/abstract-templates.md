# 摘要模板（中文 + 英文）

中文经济学论文需同时提供中、英文摘要。中文摘要 200-300 字、英文摘要 150-250 词为宜。

## 一、中文摘要的四要素

中文摘要建议遵循"问题—数据与方法—发现—含义"的四要素结构。每要素 1-3 句。**避免**：

- 第一人称（"笔者认为""本人发现"）
- 营销式语言（"具有重大意义""填补空白""开创性"）
- 堆砌缩略语和符号
- 把回归系数原样塞进摘要

## 二、通用模板（实证类）

```
[问题] 在 [中国具体制度/政策背景] 下，[研究问题] 是 [理论或现实] 中的重要议题。
[数据与方法] 本文基于 [数据来源 + 时间区间 + 样本范围]，运用 [识别策略]，
考察了 [核心变量] 对 [被解释变量] 的影响。
[发现] 研究发现：第一，…；第二，…；第三，…。机制分析表明，[机制简述]。
进一步的异质性分析显示，效应在 [子样本] 中更为显著。
[含义] 本文的研究为 [理论/政策含义] 提供了新的经验证据，
对 [应用对象，如监管部门、企业、家庭] 具有参考价值。
关键词：…
```

## 三、按识别策略分模板

### DID（政策评估类）

```
本文以 [政策名称及实施时间] 为准自然实验，基于 [数据] 构造双重差分模型，
评估了该政策对 [被解释变量] 的影响。研究发现：政策显著 [提高/降低] 了 [被解释变量]，
平均效应约为 [口径，如"0.X 个标准差"或"X 个百分点"]。
平行趋势、安慰剂检验和多种稳健性检验均支持上述结论。
机制分析表明，政策主要通过 [机制 1] 与 [机制 2] 起作用。
异质性分析显示，效应在 [子样本] 中更为突出。
本文为 [政策评价 / 优化方向] 提供了经验证据。
```

### IV（工具变量类）

```
本文以 [工具变量] 作为 [内生变量] 的工具，基于 [数据] 估计了 [核心因果效应]。
工具变量的相关性与外生性得到 [一阶段 F 值 / 过度识别检验等] 的支持。
研究发现：…；与 OLS 估计相比，工具变量估计 [增大/缩小] 了系数，说明 [遗漏变量/反向因果方向]。
机制分析表明，…。
```

### RDD（断点回归）

```
本文利用 [政策/规则导致的某一阈值] 形成的局部随机化，运用断点回归方法考察了 [核心问题]。
密度检验、协变量平衡检验和带宽稳健性检验均支持识别假设。
研究发现：在阈值附近，[处理] 使得 [被解释变量] [上升/下降] 约 [幅度]，
该结果对带宽选择、多项式阶数与核函数选择具有稳健性。
```

### 合成控制法（SC）

```
本文以 [事件/政策] 为冲击，使用合成控制法构造 [处理对象] 的反事实路径，
评估其对 [被解释变量] 的影响。…
```

## 四、英文摘要模板

英文摘要不必逐字翻译，宜重新组织成英文期刊读者熟悉的结构（动机—方法—发现—贡献）。

```
[Motivation] Against the backdrop of [Chinese institutional context],
how [policy / mechanism] affects [outcome] remains an open question.
[Method] Using [data source, sample, period], this paper applies [identification strategy]
to examine [research question].
[Findings] We find that …. The effects are robust to [robustness checks].
Mechanism analysis suggests that …. Heterogeneity analysis indicates that ….
[Contribution] This paper contributes to the literature on [topic] by [contribution]
and provides implications for [policy / practice].
Keywords: …
JEL Classification: …
```

## 五、关键词与 JEL

- 关键词 3-6 个，先具体后抽象（先方法/对象，再大主题）
- 英文摘要后建议附 JEL Classification（2-4 个），如 `G18; G21; D14`
- 关键词避免重复正文标题中的所有词

## 六、自检清单

- [ ] 是否在第一句就说清楚了"研究什么问题"？
- [ ] 数据来源、时间区间、样本范围是否齐全？
- [ ] 是否点明识别策略，而不仅是"实证研究"？
- [ ] 主要发现是否给出了量化方向（提高/降低、显著程度），但避免堆砌系数？
- [ ] 是否说明了机制？
- [ ] 是否在最后提到理论或政策含义？
- [ ] 是否避免了"具有重要意义""开创性研究""填补空白"等用语？
