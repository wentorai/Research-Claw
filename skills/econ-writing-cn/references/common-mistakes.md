# 中文经济学论文常见写作误区

> 本文列出国内学者在中文经济学论文中较常出现的写作问题。这些问题大多不在"研究设计"层面，而在"表达与组织"层面，但往往成为审稿人退稿或大修的直接原因。

## 一、引言层面

1. **"上来就讲文献"**：第一段堆砌"国外学者…国内学者…"，缺少现实问题与制度背景。审稿人需要先知道"为什么这事重要"。建议第一段写中国当下的具体问题。
2. **边际贡献写成"贡献清单"**：把每一段实证发现都列成一条贡献，导致 5-6 条贡献，反而稀释了核心贡献。建议聚焦 2-3 条，且区分"理论贡献"与"经验贡献"。
3. **滥用"填补空白""开创性"**：审稿人会对此类自夸语高度敏感。改为"对 … 文献提供了新的经验证据"。
4. **缺少识别策略一句话陈述**：引言应有一句"本文利用 … 作为外生冲击/工具/断点 …"。读者不应读到第 4 节才知道你的识别。

## 二、文献综述

1. **罗列式综述**：A 学者发现 X，B 学者发现 Y，C 学者发现 Z…缺乏对比与结构。建议按"主题—争议—本文位置"组织。
2. **只引用国内文献或只引用国外文献**：国内顶级期刊期望两类文献兼具。
3. **引用过于陈旧**：5 年内的工作比例过低会让审稿人怀疑作者文献跟踪不及时。
4. **把综述写成"谁的研究最相关就批评谁"**：中文学界相对小，过度批评近作易引发审稿矛盾。要点是"区别"而非"否定"。

## 三、数据与变量

1. **变量定义模糊**：例如"企业规模 = 资产对数"——但样本中是否取对数、是否+1、是否 winsorize、是否单位化？建议附变量定义表。
2. **样本筛选不交代**：剔除 ST、剔除金融业、剔除资不抵债、剔除上市未满 X 年等条件需明确。
3. **数据来源含糊**：CSMAR、Wind、CCER、企查查、税务局微观数据库等需明确版本与下载日期。
4. **缺少描述性统计极值检查**：审稿人常会问"为什么资产负债率均值 > 1"。Winsorize 1%/99% 是惯例，需写明。

## 四、计量与结果

1. **方程不写完整**：只写 `Y = a + bX + e` 而不交代下标 `i, t`、固定效应、聚类层级。
2. **聚类标准误粗糙**：默认聚类到企业可能不够，需考虑省份×年份等多维聚类，并讨论聚类层级敏感性。
3. **回归表过于拥挤**：把 10 个控制变量都列出来挤满半页。建议主表只列核心变量，控制变量在脚注或附录。
4. **滥用 ***、**、\***：星号水平要在表注里说明（10%、5%、1%）。不要每张表都解释一次。
5. **机制分析变成"分组回归"**：仅分组对比不能直接证明机制，需要在被解释变量上做替换或在中介变量上做检验，且与机制故事一一对应。
6. **稳健性堆数量、不解释**：每一个稳健性检验都应在文中用 1-2 句解释"针对哪个担忧"。

## 五、政策含义

1. **政策建议空泛**：如"建议加强监管""建议优化制度"——这种话谁都会写，对读者无新增信息。建议结合本文实证发现，落到可执行层面（哪类政策工具、面向哪类对象、力度多大）。
2. **政策建议超出实证支持**：基于上市公司样本得出的结论，不宜直接推到中小微企业；基于 2000-2015 年数据的结论，不宜直接推到当下。
3. **重复正文已说的内容**：政策建议不是结论的复读机。

## 六、语言与排版

1. **口语化**：避免"非常""特别""一下子""很大很大"。
2. **比喻过度**："如同雨后春笋""势如破竹"等不宜出现在学术文本。
3. **被动语态过多**：中文论文中"被发现""被认为"过多会让文本沉闷。
4. **图表说明不独立**：表注/图注应在没有正文的情况下也能读懂，包含数据来源、样本量、估计方法、聚类层级、显著性水平。
5. **参考文献格式不统一**：作者格式（中外文混排）、年份位置、起止页码缺失等是常见扣分项。
6. **过多英文缩略语**：首次出现需中文全称 + 英文缩写。

## 七、伦理与合规

1. **未声明利益冲突**：尤其是受公司、行业协会资助或样本由企业提供时。
2. **未交代数据使用授权**：使用税务、银行、医疗等敏感微观数据需说明授权来源。
3. **未公开复现资料**：越来越多期刊要求提交复现包（数据/代码）。即使期刊未要求，也建议在脚注注明"如需数据与代码，可向通讯作者索要"。
4. **AI 辅助未声明**：大语言模型用于语言润色、代码辅助时，建议在致谢或脚注中说明使用情况。

## 八、自我检查清单

阅读自己稿件时按以下顺序快速过一遍：

1. 引言第一句：是否说了"中国当下的什么问题"？
2. 边际贡献：是否限定在 2-3 条？是否区分理论与实证？
3. 识别策略：是否一句话讲清？识别假设是否在文内显式陈述？
4. 主表：是否一眼能看出方法、样本、聚类、显著性？
5. 政策建议：是否具体到工具与对象？
6. 参考文献：格式是否统一？
7. 摘要：四要素是否齐全？是否避免了自夸语？
