# NSFC 真实学科范例（5 学科 × 4 维度）

本文件为 5 个不同学部的 NSFC 申请书提供**具体学科语境的写作范例**。研究方向均标注 "(假设性，仅用于示意)"——但学科语言、数据范式、方法论框架真实可用。每个学科给出：

1. 优秀立项依据（漏斗结构）
2. 优秀关键科学问题凝练
3. 优秀创新点 3+1（理论 / 方法 / 应用 + 概念）
4. ❌ 反面例子 + 修改建议

> 范例研究方向均为示意性构造，不指代任何真实在投或已结题项目。学科语言基于公开发表论文与各学部近 5 年评审综述提炼。

## 范例 1 — F02 信息工程：图神经网络在金融风控的可解释性 (假设性，仅用于示意)

### 1.1 优秀立项依据（漏斗结构 ~180 字）

> 随着移动支付与小额信贷交易日均规模突破 10 亿笔，基于图神经网络（Graph Neural Networks, GNN）的金融风控模型已成为头部银行欺诈识别主力——但模型决策的"黑箱性"导致监管审查与客户申诉时无法溯源，已成为《商业银行模型风险管理指引》（2022）落地的关键障碍。当前 GNN 可解释性研究主要沿两条路径：基于扰动的 GNNExplainer（Ying et al., NeurIPS 2019）与基于梯度的 PGExplainer（Luo et al., NeurIPS 2020）。然而，金融图具有**异质性 + 时序漂移 + 强不平衡**三重特性，现有方法在反洗钱场景的解释稳定性指标 (Fidelity-) 仅 0.42——远低于工业部署阈值 0.75。**本课题旨在通过引入因果不变性约束，将异质金融图的 GNN 解释 Fidelity- 提升至 0.80 以上**，为可解释 AI 在监管科技 (RegTech) 落地提供方法基础。

### 1.2 优秀关键科学问题（1-3 句）

> **关键科学问题**：在异质性 + 时序漂移的金融图场景下，如何通过因果不变性约束，构造**对子图扰动稳定且对人工审计可读**的 GNN 解释器？具体分解为：(1) 如何形式化定义"金融图的解释稳定性"以同时满足 Fidelity- 与 Sparsity 约束？(2) 在样本极度不平衡（欺诈样本占比 < 0.1%）下，因果不变性约束的可识别性条件是什么？

### 1.3 优秀创新点 3 + 1

- **理论创新**：首次将 invariant risk minimization (IRM) 框架引入异质图解释器，证明在金融图典型分布漂移下解释稳定性的可识别性条件（定理 1）。
- **方法创新**：提出 CausalGNN-Ex 算法，结合子图采样 + 反事实预测 + IRM 损失，将解释 Fidelity- 从基线 0.42 提升至 ≥ 0.80。
- **应用创新**：在某大型城商行真实反洗钱图（节点 1.2 亿、边 8 亿）上验证可部署性，并形成可对接央行监管沙盒的解释报告模板。
- **概念创新**：提出 "regulator-readable explanation" 评价范式——可解释性不仅是数学指标，更是**监管者 30 秒可读 + 可审计**的工程属性。

### 1.4 ❌ 反面例子 + 修改建议

❌ **反面**：
> 本项目研究图神经网络在金融领域的应用。我们将提出新方法提升 GNN 性能。预期在多个数据集上达到 SOTA，发表 CCF-A 论文 3 篇。

**为什么差**：
- "图神经网络在金融领域的应用"过于宽泛，不是关键科学问题；
- "提升 GNN 性能"——性能哪个维度？基线是什么？
- "发表 CCF-A 论文 3 篇"——结果导向的 KPI 不是学术问题；
- 没有可证伪的假设。

**修改建议**：把"应用"窄化到**异质图 + 不平衡 + 时序漂移**这一三元组下的可解释性，把"性能"具体化为 Fidelity- ≥ 0.80，把"3 篇 CCF-A"换成 Aim 1 / 2 / 3 的可验证目标。

## 范例 2 — G02 工商管理：数字化转型对制造业全要素生产率的双刃剑效应 (假设性，仅用于示意)

### 2.1 优秀立项依据 (~190 字)

> 数字化转型已被《"十四五"数字经济发展规划》定位为制造业升级核心引擎。现有文献多基于"技术-组织-环境"框架报告**正向**生产率效应（Brynjolfsson & McElheran, AER 2016；许宪春等，《管理世界》2022），但 2019-2023 年中国 A 股制造业上市公司财报数据显示：实施 ERP/MES 转型的样本中，**约 38% 在转型 18 个月后全要素生产率 (TFP) 反而下降**——这一"数字化生产率悖论"在工业互联网试点企业中尤为显著。现有研究的局限在于：(1) 多用横截面或短面板，无法识别"动态调整成本"；(2) 把"数字化"作为同质二元变量，忽视技术异质性。**本课题拟基于 2015-2024 年规上制造业企业普查与海关贸易匹配数据，运用差分中的差分 (DID) + 事件研究法识别数字化转型的动态生产率效应**，并探讨"组织协同能力"作为关键调节变量的边界条件。

### 2.2 优秀关键科学问题

> **关键科学问题**：数字化转型对中国制造业全要素生产率的**短期—长期非线性效应**的微观机制是什么？具体分解为：(1) 转型后 TFP 的 J 型曲线（先降后升）的动态调整周期与"低谷深度"由哪些企业异质性变量决定？(2) "组织协同能力"（员工技能匹配 + 流程数字化适配）在多大程度上调节这一效应？

### 2.3 优秀创新点 3 + 1

- **理论创新**：拓展 Bresnahan-Brynjolfsson-Hitt (2002) 的"数字化-组织协同"互补性理论，将"动态调整成本"内生化进生产函数。
- **方法创新**：构造"数字化转型强度"连续度量（基于年报文本 + 招聘岗位 + IT 资本 / 总资产三角识别），克服现有"二元数字化"度量的失真。
- **应用创新**：识别"协同能力低 + 资本密集型"企业的转型门槛，为工信部"专精特新"与中小企业数字化补贴政策提供精准画像。
- **概念创新**：提出 "Productivity J-curve" 范式，区分"数字化的吸收期"与"价值释放期"。

### 2.4 ❌ 反面例子 + 修改建议

❌ **反面**：
> 本项目研究数字化对企业绩效的影响。运用面板数据回归分析数字化的影响因素与作用机制。预期发表 SSCI 论文若干篇。

**为什么差**：
- "影响"未指明方向、机制、边界条件；
- "影响因素与作用机制"——典型的"研究 XX 影响因素"宽泛表述（NSFC 退稿信号 #1）；
- 缺因果识别策略；
- 结果导向 KPI。

**修改建议**：把"影响"具体到**TFP 的 J 型动态曲线**；把"影响因素与作用机制"换成**1 个核心 hypothesis + 2 个可验证 sub-hypothesis**；增加 DID + 事件研究的因果识别。

## 范例 3 — H 临床医学：肠道菌群失衡介导抑郁症认知障碍 (假设性，仅用于示意)

### 3.1 优秀立项依据 (~190 字)

> 抑郁症全球患病率 ~5%，其中 50-60% 患者伴随认知功能障碍 (cognitive dysfunction in MDD, MDD-CD)；现有 SSRI/SNRI 抗抑郁治疗对核心情绪症状有效，但**对认知障碍缓解率仅 ~30%**——这一治疗缺口已被 NEJM (2023) 列为下一代精神药物研发关键靶点。近年微生物-肠-脑轴 (MGB axis) 研究提示：抑郁症患者粪菌多样性下降，且产丁酸菌属 (e.g., *Faecalibacterium*) 丰度降低与海马依赖性认知障碍相关 (Cryan et al., Cell 2019; Zheng et al., Nat Microbiol 2022)。然而，现有人体队列多为横截面 + 16S rRNA 浅测序，无法识别**菌群-代谢物-小胶质细胞-海马回路**的因果链。**本课题拟基于 600 例 MDD 患者前瞻性队列 + C57BL/6J 小鼠粪菌移植 (FMT) 模型，整合宏基因组 + 靶向代谢组 + 小胶质细胞单细胞测序**，识别介导认知障碍的关键菌-代谢物-神经免疫节点，为益生菌辅助治疗提供机制证据。

### 3.2 优秀关键科学问题

> **关键科学问题**：肠道菌群失衡通过哪些代谢物-免疫信号通路调节海马小胶质细胞活化，进而介导抑郁症认知障碍？具体：(1) 哪些菌属及其代谢物（短链脂肪酸 / 色氨酸 / 胆汁酸）是必要充分的因果节点？(2) 小胶质细胞 TREM2 信号在该通路中的调控权重？

### 3.3 优秀创新点 3 + 1

- **理论创新**：提出"菌群-代谢-免疫-认知"四层因果模型，超越现有"菌群-情绪"二层框架。
- **方法创新**：建立**人体队列 + FMT 小鼠 + 单细胞解析**的三层因果验证流程，避免现有研究"相关 ≠ 因果"的局限。
- **应用创新**：筛选 2-3 株可临床转化的产丁酸菌株原型，为辅助 SSRI 治疗的菌群干预药械提供前体。
- **概念创新**：提出 "MDD-CD endotype" 分型——按肠道菌群特征分层抑郁症认知障碍患者，奠定精准精神病学基础。

### 3.4 ❌ 反面例子 + 修改建议

❌ **反面**：
> 本项目研究肠道菌群与抑郁症的关系。检测患者粪便菌群组成，与健康对照比较。预期发现差异菌种，发表 SCI 论文 5 篇。

**为什么差**：
- "关系"——相关研究而非机制研究，H 学部直接给 C；
- 缺人体 + 动物 + 多组学的因果验证；
- 缺临床转化落点；
- 结果是"差异菌种"，而非可干预的靶点。

**修改建议**：把"关系"具体到**因果通路**；增加 FMT 小鼠模型 + 单细胞解析；明确临床转化（菌株原型 / 干预药械）。

## 范例 4 — B 化学：单原子催化剂选择性氧化 (假设性，仅用于示意)

### 4.1 优秀立项依据 (~180 字)

> 苯甲醇选择性氧化制苯甲醛是香料 / 医药中间体年产值 > 80 亿美元的核心工艺。现有贵金属负载催化剂 (Pd/C, Ru/Al₂O₃) 在 80°C / 1 atm O₂ 条件下苯甲醛选择性 ~85%，但**贵金属用量 ~5 wt%、过氧化产物苯甲酸副产 ~10%**——成本与分离工艺已成为绿色化学产业化瓶颈。单原子催化 (Single-Atom Catalysis, SAC) 通过 100% 原子利用率 + 配位环境精确调控被认为是下一代催化剂方向 (Zhang et al., Nature Catalysis 2023)，但当前 N-doped carbon 载体的 Pt/Pd SAC 在该反应中**苯甲醛选择性仅 78%**——配位结构与产物分布的构效关系尚不明晰。**本课题拟通过 Co-N₄ 单原子位点的轴向配位调控**，将苯甲醛选择性提升至 ≥ 95%，并结合原位 EXAFS + DFT 揭示选择性来源。

### 4.2 优秀关键科学问题

> **关键科学问题**：在 Co-N₄ 单原子催化剂上，**轴向第六配位原子（O / S / N）如何通过电子结构调控决定苯甲醇 → 苯甲醛 vs 苯甲酸的选择性分岔？** 即：选择性的电子描述符 (electronic descriptor) 是什么、可不可调？

### 4.3 优秀创新点 3 + 1

- **理论创新**：首次建立 Co-SAC 轴向配位电子态与苯甲醛选择性的定量构效关系（描述符：Co 3d-band center vs O₂ 吸附能）。
- **方法创新**：开发"轴向配位精确植入"合成法（Co-N₄-X，X = O/S/N），实现单原子位点 ≥ 95% 配位精度。
- **应用创新**：在中试反应器（50 g 催化剂 / 5 L 反应釜）实现连续 200 h 选择性 ≥ 95%、贵金属用量降至 0.5 wt%，与某香料企业合作启动 100 kg/d 试生产。
- **概念创新**：提出 "axial coordination engineering" 范式——超越现有面内配位环境调控的单原子催化设计原则。

### 4.4 ❌ 反面例子 + 修改建议

❌ **反面**：
> 本项目合成新型单原子催化剂用于氧化反应，研究其催化性能。预期获得高活性催化剂，发表 JACS 等高水平论文。

**为什么差**：
- "新型"无定义；"氧化反应"无具体目标产物；
- "高活性"无量化基线；
- 缺机理研究——B 学部偏好"机理清晰"；
- 结果导向。

**修改建议**：把目标反应窄化到**苯甲醇→苯甲醛**；把"高活性"换成**选择性 ≥ 95% + 贵金属 0.5 wt%**；增加 EXAFS + DFT 机理研究。

## 范例 5 — A 数学：高维统计推断的非渐近置信区间 (假设性，仅用于示意)

### 5.1 优秀立项依据 (~170 字)

> 高维统计 (p ≫ n) 在基因组关联分析、金融因子建模等场景已成主流。Lasso / SCAD / MCP 等正则化估计的**点估计**理论已较成熟 (Bühlmann & van de Geer, 2011)，但**置信区间构造**（即如何在高维下做可靠的假设检验）仍是开放问题：现有 de-sparsified Lasso (van de Geer et al., AoS 2014) 给出渐近正态结果，但**有限样本覆盖率在 n = 200, p = 1000, s₀ = 20 时仅 0.82**（名义 95%），远低于实际可信区间需求。**本课题拟通过引入自适应数据分割 + 经验贝叶斯校正，建立非渐近 (non-asymptotic) 覆盖率 ≥ 0.93 的高维置信区间构造**，并证明在亚高斯设计下的最优率。

### 5.2 优秀关键科学问题

> **关键科学问题**：在高维稀疏线性模型下，**有限样本意义下覆盖率名义可达 (1-α) 的置信区间构造，其样本复杂度的最优阶是什么？** 即：min n s.t. P(β ∈ CI) ≥ 1-α 在 (s₀, p, σ) 上的下界？

### 5.3 优秀创新点 3 + 1

- **理论创新**：建立高维置信区间的**有限样本下界**（定理 1）与可达性（定理 2），关闭现有渐近-有限样本之间的 gap。
- **方法创新**：提出 "Adaptive Splitting Empirical Bayes" (ASEB) 算法，在 n = 200, p = 1000 模拟下覆盖率 0.93+。
- **应用创新**：将方法应用于 UK Biobank 基因关联分析（自然 p ≈ 10⁶, n ≈ 5×10⁵），识别 SNP 显著性的可靠置信区间。
- **概念创新**：提出"非渐近最优性"作为评价高维推断方法的新范式，区别于经典渐近最优。

### 5.4 ❌ 反面例子 + 修改建议

❌ **反面**：
> 本项目研究高维统计的应用。运用机器学习方法分析大数据。预期得到新结果。

**为什么差**：
- A 学部最忌讳"应用"导向——数学项目要有**自身价值**的定理；
- "机器学习方法 + 大数据"是非数学语言，A 学部评审会判跨界；
- "新结果"完全空洞。

**修改建议**：把目标定为**非渐近覆盖率最优率定理 + 可达算法**；用纯数学语言（亚高斯设计、稀疏度 s₀、收敛阶）表达；应用部分（UK Biobank）只作为示意。

## 通用排版与写作约定

- **每个学科的语言**要符合其学部评审综述的"母语"——见 `references/reviewer-psychology.md` 第五节；
- **数字优先**：宁可写"选择性 95%"也不要写"高选择性"；
- **基线必须明确**：每个改进都要有"vs 现有方法 X% → 我们 Y%"对照；
- **创新点 3 + 1**：理论 / 方法 / 应用 必备，第 4 个"概念创新"是加分项（重点 / 杰青 / 重大 几乎必备）；
- **关键科学问题"凝练"**：1-3 句，不超过 3 句；动词以"为什么 / 如何 / 是什么"打头，避免"研究"。
