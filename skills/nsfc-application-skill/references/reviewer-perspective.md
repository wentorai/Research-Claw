# NSFC 评审专家视角：函评 + 会评

NSFC 实行 **"函评 + 会评"双轮制度**。理解评审视角是申请书写作的第一性原则 — 写作目标不是"打动自己"，而是"通过 5 位小同行函评 + 学科组会评的双重过滤"。

## 一、函评（通讯评议）

### 1. 评审主体

- 由 NSFC 科学处从专家库中按学科代码 + 关键词派送
- 每份申请书 **派送 5 位** 小同行
- 评审为匿名、网络化操作（ISIS 系统）

### 2. 评审维度（公开打分项，5 维）

1. **科学价值 / 学术意义**：研究问题对学科的重要性、对国家需求的对应度
2. **创新性**：与已有工作相比的"新意"（新思路、新方法、新数据、新视角）
3. **研究方案的可行性与合理性**：技术路线、方法选择、预实验、可达成性
4. **申请人 / 研究团队基础**：代表作、与课题的连续性、团队构成
5. **预算合理性**：与研究内容是否匹配

### 3. 评审等级（典型 5 档）

- **A. 优先资助**
- **B. 可资助**
- **C. 可不资助**
- **D. 不予资助**
- **E. 未达基本要求 / 撤回**

通常 **5 份评审至少需要 3-4 份 A/B 才能进入会评**。1 份 D 几乎"一票否决"。

### 4. 函评专家阅读模式

- **平均阅读时间**：每份 30-60 分钟
- **首先看**：摘要 / 立项依据首页 / 创新点 / 申请人代表作
- **其次看**：研究内容 + 关键科学问题 + 技术路线图
- **最后核**：可行性 + 预算（粗看）

**写作启示**：摘要 / 立项依据前 1 页 / 创新点 / 代表作这 4 处必须"一击命中"。

## 二、会评（学科组评审）

### 1. 评审主体

- 学科组由 NSFC 聘请的 **会评专家**（通常 10-20 人）组成
- 仅审议函评得分较高（如 4A1B 及以上）的项目
- 会评在北京集中召开，每个项目由"主审 + 副审"现场介绍并答辩

### 2. 会评关注点

- 函评中存在分歧的项目（如 3A2C）
- 创新性在 4A 中是否真的"突出"
- 团队结构是否合理（重点 / 重大类）
- 当年学科指南方向匹配度（重大研究计划类）

### 3. 会评决定的 3 类结果

- 通过资助
- 不通过资助
- "降档"资助（如重点降为面上 — 较少见）

## 三、加分项

| 加分点 | 写法示例 |
| --- | --- |
| 真正的预实验数据 | 在"可行性分析"放 1 张预实验图 + 简要数据 |
| 高水平代表作（一作 / 通讯）| 代表作放在"研究基础"前 3 篇，注明对应研究内容 |
| 与课题方向高度连续 | 用 1 段写"申请人 5 年聚焦 XX 方向，本项目是逻辑延伸" |
| 清晰的技术路线图 | 1 张图说清楚 R1→R2→R3 与产出 |
| 明确的可证伪科学问题 | "本项目拟检验：当 X 满足 …… 条件时，Y 是否仍 ……" |

## 四、退稿信号（函评 D / E 的高频原因）

- **创新性不清晰**：创新点写成"研究内容复述"
- **关键科学问题宽泛**：如"研究 XX 的影响因素"
- **方案空洞**：缺技术路线图、缺方法细节、缺预实验
- **代表作与课题脱节**：申请人 5 年代表作多在另一方向
- **预算异常**：设备费占比 > 50%、劳务费 = 0、版面费过高
- **跨界跳跃**：从生物突然申请管理科学；从计算机突然申请化学
- **抄袭嫌疑**：被检测出与公开来源（含已结题项目报告）大幅雷同

## 五、写作 → 评审 反向工程

把申请书写作目标设定为：

1. **30 秒内能让小同行答出**：你做什么？为什么有意义？
2. **3 分钟能让小同行答出**：你的关键科学问题、研究方案、创新点
3. **30 分钟能让小同行答出**：你能做成（基础匹配 + 可行性硬证据）

如果上述任何一项做不到，就需要回炉重写对应章节。

## 六、注意事项

- 上述评分维度、档次、流程是 NSFC 公开制度的稳定部分；**具体表述以当年《国家自然科学基金资助项目评审办法》为准**。
- 函评名单与会评名单全程匿名，禁止打听评审专家。
- 申请书与评审意见均纳入 NSFC 学风诚信档案。
