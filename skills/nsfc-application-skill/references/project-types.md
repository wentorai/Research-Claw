# NSFC 8 类常见项目对比

下表为 NSFC 公开公布的常见 8 类项目的核心参数与定位差异。**资助强度、年限、申请人条件按当年《项目指南》为准**，本表仅给定通常稳定的相对关系，不写具体金额。

## 一、按"自由探索"主线分类

### 1. 青年科学基金项目（青年项目）

- **定位**：扶持青年科研人员独立开展研究
- **申请人条件**：男 ≤ 35 岁、女 ≤ 40 岁；具有高级职称或博士学位
- **次数限制**：仅可申请 1 次（含获批）
- **资助强度**：在 4 类一般项目中最低；周期一般 3 年
- **评审重点**："潜力 > 完整性"；代表作 1-2 篇即可，但要"自己一作或通讯"

### 2. 面上项目

- **定位**：自由选题、自由探索（受众最广）
- **申请人条件**：高级职称 或 博士学位 + 2 名同行高级职称推荐
- **资助强度**：周期 4 年
- **评审重点**：科学问题清晰 + 研究方案严谨 + 申请人有连续工作积累
- **典型受众**：副高 / 副教授；博士后已出站

### 3. 地区科学基金项目

- **定位**：扶持西部及特定地区科研力量
- **申请人条件**：依托单位须在限定地区
- **资助强度**：低于面上、高于青年；周期 4 年

## 二、按"重点 / 重大方向"分类

### 4. 重点项目

- **定位**：在已有积累基础上做深入、系统性研究
- **申请人条件**：通常需有正高 + 已结题面上 / 杰青等
- **资助强度**：远高于面上；周期 5 年
- **评审重点**：方向凝聚 + 团队结构 + 已有成果体系

### 5. 重大项目 / 重大研究计划

- **定位**：服务国家战略需求，自上而下设方向
- **申请方式**：按当年发布的"指南方向"申请
- **资助强度**：项目级最高之一；周期 4-5 年
- **评审重点**：方向匹配度 + 团队整合度 + 成果可期

## 三、人才类项目

### 6. 国家杰出青年科学基金（杰青）

- **定位**：在前沿方向取得创新性成果的领军人才
- **申请人条件**：男 ≤ 45 岁、女 ≤ 48 岁
- **次数限制**：每人仅 2 次申请机会
- **评审重点**：代表作"硬度"+ 已有原创性贡献 + 未来 5 年方向

### 7. 优秀青年科学基金（优青）

- **定位**：杰青预备队
- **申请人条件**：男 ≤ 38 岁、女 ≤ 40 岁
- **评审重点**：代表作 + 独立研究方向

### 8. 创新研究群体项目（创新群体）

- **定位**：依托杰青 / 长江等已有人才形成稳定研究群体
- **申请方式**：以群体名义申请，强调"研究方向凝聚 + 人才梯队 + 已有合作"
- **资助强度**：与重大相当；周期 6 年（含续期）

## 四、国际合作类（NSFC-XXX）

NSFC 与多国基金机构有双边协议（如 NSFC-DFG / NSFC-RGC / NSFC-NRF / NSFC-JSPS / NSFC-RS / NSFC-FWF 等）。

- **申请方式**：中外双方各自向本国基金机构提交协调一致的申请书
- **评审**：双边联合评审（含两边小同行）
- **要点**：合作必要性、双方分工、联合产出（论文 / 联合培养 / 互访）

## 选择建议

- **首次申请人** → 优先青年项目，做减法、聚焦 1 个关键问题
- **已结题青年 / 副高** → 面上项目；将青年的延续性体现出来
- **已结题面上 + 有"硬"代表作** → 视方向重要程度选择 重点 / 优青 / 杰青
- **承担重点 / 杰青多年** → 创新群体 / 重大项目
- **有海外稳定合作** → 双边国际合作

## 注意事项

- 同一年同一申请人在 NSFC 系统内的 **限项规则** 较为复杂（在研项目数 + 申请数总和有上限）。务必在申报前查阅当年《项目指南》"限项申请规定"章节。
- 各类项目的具体金额、年龄、字数上限**每年微调**；本文件不写具体数字，请以官网为准。
