# NSFC 学部与申请代码

NSFC 按学科分为 **9 大科学部**（含 2 个专设管理部门），项目申请按"学部—一级代码—二级代码—三级代码"层级匹配。**写错代码 = 函评直接被退回 / 错送评审**，是首要核对项。

> 注：自 2021 年代码改革以来，各学部三级代码做过较大调整。**本文件仅给出 7 大常见学部及部分二级学科示例（公开信息），不写完整三级代码 — 请以 NSFC 官网当年公布的《代码表》为准**。

## 三、代码选择 4 条原则

1. **先定学部**：跨界研究先想清楚"主要学术贡献归属哪个学部" — 评审专家来自该学部
2. **二级代码 = 函评专家库**：选错二级代码 = 拿到评审专家不是小同行，意味着退稿风险大幅上升
3. **三级代码精确化**：二级代码下还有三级代码（如 G0301 微观经济学、G0302 宏观经济学），尽量准确
4. **可附 2-3 个研究方向关键词**：申请系统允许填关键词，便于科学处加权派送

## 四、跨学部 / 交叉申请

近年 NSFC 明确鼓励交叉研究，**交叉科学部** (T) 接受跨学部研究。但需注意：

- 交叉科学部 ≠ "什么都能投"；申请书要明确"交叉的科学问题"
- 选交叉科学部的项目，函评专家可能横跨多学部，论证要兼顾不同领域读者
- 如果研究只是"用 A 学部的方法做 B 学部的问题"，多数情况下还是选 B 学部更合适

## 五、注意事项

- **本文件二级学科代码示例为公开常识，三级代码每年微调**。最终代码以 NSFC 官网公布的《XXX 年度国家自然科学基金项目申请代码》为准。
- 部分代码改革后已合并或重新划分（如生命科学部、医学科学部），切勿凭印象套用旧代码。
- 申请前可登陆"国家自然科学基金网络信息系统 (ISIS)"查看本年度开放的代码列表。
