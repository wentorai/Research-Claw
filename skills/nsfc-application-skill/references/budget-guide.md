# NSFC 预算编制规范

NSFC 自 2021 年起对面上 / 青年 / 地区项目实行 **"包干制"**（不再编制预算明细），其他项目仍执行**预算制**（按科目编列）。本文件介绍预算制下的科目要点。**具体上限、比例每年微调，请以当年《项目指南》"经费预算"章节为准**。

## 一、总体结构

预算 = **直接费用** + **间接费用**

- **直接费用**：用于项目研究的实际开支，由申请人按 9 个科目分别编列
- **间接费用**：依托单位用于补偿与本项目相关的间接消耗（含绩效），按一定比例计提

## 二、直接费用 9 大科目

### 1. 设备费

- **小型设备购置**：单台 ≤ 50 万元（具体上限按当年规定）
- **设备试制 / 改造 / 升级**
- **设备租赁**
- **要点**：设备费一般不超过总预算的 30%；**青年项目压缩到 5% 以内更稳妥**。购买大型设备需说明本单位无现有可替代的理由。

### 2. 材料费

- 实验材料、化学试剂、低值易耗品、小工具、参考资料
- 要按"用途 + 大致数量 + 单价区间"做支撑性说明，避免"实验材料 10 万元"一行

### 3. 测试化验加工费

- 委托外单位的测试、化验、检测、加工
- 要写清楚委托单位类型（如"高校测试中心"）和大致工作量

### 4. 燃料动力费

- 大型设备运行、风洞、专用计算服务器电费等
- 多数文科 / 计算类项目此项可填 0 或较少

### 5. 差旅 / 会议 / 国际合作与交流费（合并科目）

- 国内调研 + 学术会议 + 国际合作交流
- 国际交流要写"派出 X 人次 + 来访 Y 人次"，**不可与差旅混淆**

### 6. 出版 / 文献 / 信息传播 / 知识产权事务费

- 论文版面费、出版费、专利申请与维护费、数据库订阅、文献检索

### 7. 劳务费

- **可全额发放**给：在校研究生、博士后、未纳入单位编制的项目聘用人员
- **不可发放**给：申请人本人、单位编制内的合作者
- **无固定上限**，但需"合理"。劳务费占比过高（>30%）会被审计关注

### 8. 专家咨询费

- 聘请咨询专家费用，按"次数 × 标准"列示
- **不得发放**给参与本项目的研究人员；多数申请书填 0 或较少

### 9. 其他

- 通讯、运输等无法归入上述科目的杂项

## 三、间接费用

按依托单位所属性质分档计提（一般 ≤ 30%，含项目组绩效）：

- **科研单位 / 高校**：通常 ≤ 30%（500 万元以下部分）
- **企业**：比例较低
- **绩效**：从间接费用中支付给项目组成员，激励性质

## 四、预算编制 5 条要点

1. **预算与研究内容互相印证**：研究内容里写"开展 3 次田野调查"，差旅费就要列出 3 次的人次与地点估算
2. **不要故意压低**：预算压低反而显得"想做的事不够"
3. **不要堆设备**：评审会问"为什么本单位没有可替代设备"
4. **劳务费要"养得起人"**：研究生津贴按本地标准估算（如硕士 800-2000/月、博士 2000-3500/月、博士后 1.5-3 万/月）
5. **国合 / 派出来访要分清**：合作交流费 ≠ 差旅费

## 五、包干制（青年 / 面上 / 地区）

包干制下，申请人**不需要编制预算明细**，仅需说明经费用途的大致结构（如材料、劳务、差旅、出版各占比例）。但仍需注意：

- 包干制并非"无监管" — 经费使用仍须符合财务制度
- 在"经费预算说明"栏，建议简述"主要用于材料 X% / 劳务 Y% / 差旅 Z% / 其他"

## 六、常见预算问题

- 设备费过高（占比 > 30%）
- 劳务费写 0 — 评审会怀疑"研究生从哪儿来"
- 国际合作交流费与差旅费混填
- 测试化验加工费与材料费重复计列
- 单价 50 万以上设备未走"专家论证"流程

## 注意事项

- 预算金额一旦获批，调剂权限按当年财政部 / NSFC 联合规定执行；动用前请咨询所在单位科研处。
- 本文件仅介绍稳定性较高的科目结构，**具体限额请以当年《项目指南》"经费预算"章节及财务部门解读为准**。
