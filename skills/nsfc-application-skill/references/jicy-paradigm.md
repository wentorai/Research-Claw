# 杰青 / 优青范式专题（与面上的差异）

> 注：本文件描述的"杰青范式"是基于公开材料整理的**写作范式总结**——包括基金委 OER 公开的"杰青讲座"系列、《国家杰出青年科学基金完整申请书选编》公开本、以及多位获资助专家在公开渠道（学校官网、公开博客、公开讲座）的分享归纳。**不引用任何具体获奖人姓名 / 申请书原文**——仅总结其共性范式。

---

## 一、为什么"杰青范式"与"面上范式"显著不同？

| 维度 | 面上 / 青年项目 | 杰青 / 优青项目 |
| --- | --- | --- |
| **评审重心** | 项目本身（创新性 + 方案 + 基础） | **人** + 项目作为方向延伸 |
| **代表作要求** | 5-10 篇，与课题相关 | 5 篇，**全部一作 / 通讯**，标志性 |
| **方向连续性** | 5 年聚焦的"加分项" | 5 年聚焦的"刚需"——方向飘移直接出局 |
| **研究方案深度** | 完整可执行 | 完整 + **5 年蓝图 + 学科地位** |
| **创新点要求** | 思想 / 方法 / 应用 3 条 | "3+1"：3 条 + 1 个**项目级标志性概念** |
| **答辩** | 无 | **有现场答辩**（杰青二面 + 优青部分） |
| **资助强度** | 50-70 万 / 4 年（面上） | 400 万 / 5 年（杰青）；200 万 / 3 年（优青） |

**核心差异**：杰青看的不是"这个 4 年项目能不能做成"，而是"**这个人值不值得 NSFC 押注 5 年作为领域领军人才**"。

## 二、杰青范式的 5 个核心写作策略

### 策略 1：以"申请人 5 年聚焦方向"统领立项依据

不同于面上从"领域问题"开始，**杰青立项依据的段 1 通常是"申请人 5 年聚焦方向 + 标志性突破"**。

**句式模板**：
> "申请人自 20XX 年以来持续聚焦 [方向 X] 研究，取得 3 项标志性突破：(1) ……；(2) ……；(3) ……。本项目以申请人 5 年工作为基础，向 [方向 Y] 进一步延伸，聚焦 ……。"

**关键点**：
- 标志性突破需有**国际同行认可**（顶刊论文 / 大会主旨报告 / 国际奖项 / 高被引论文）
- "向 [方向 Y] 延伸"必须是**逻辑延伸**，不是新方向；否则评审会判"赶热点"

### 策略 2：构建项目级"标志性概念"（命名权）

杰青范式的"创新点"几乎都会给项目一个**可记忆的标签 / 概念 / 框架名**。

**示例（虚构）**：
- "**双向耦合假说 (Bidirectional Coupling Hypothesis, BCH)**"
- "**动态注意力路由 (Dynamic Attention Routing, DAR)**"
- "**算力—精度帕累托前沿 (Compute-Accuracy Pareto Frontier)**"
- "**用户黏性陷阱 (Stickiness Trap, ST)**"

**为什么有效**：
- 评审专家在 30 秒内**记住你这个项目**
- 命名权 = 学科话语权——这是杰青评审看重的"领军人才"信号
- 命名后的概念可以"反复出现"在创新点 / 研究内容 / 路线图 / 摘要中，强化记忆

**写法**：
- 命名 + 全称 + 缩写 ：在创新点首条 + 摘要 + 立项依据末段重复出现
- 概念要可"展开为可验证假设"（H1, H2, H3）——避免只是"漂亮名字"

### 策略 3：5 年蓝图 + 学科地位定位

杰青申请书需在**研究方案末段或单独段落**中给出：

1. **本项目 5 年内的 3 个里程碑**（M1, M2, M3）
2. **5 年后申请人在该领域的目标定位**——"成为 X 方向的国际重要研究者 / 形成 Y 方向的有影响力学派"
3. **国际合作 / 学科组织参与计划**（如组织国际会议、Editorial Board、加入国际工作组）

**句式**：
> "本项目预期 5 年内完成 3 个标志性里程碑：(M1) ……；(M2) ……；(M3) ……。5 年期满，预期申请人在 [方向 X] 形成具有国际影响力的研究方向，建立 …… 学派 / 平台 / 国际合作网络。"

### 策略 4：代表作的"硬度"与"故事线"

杰青代表作的标准（公开范式整合）：

- **数量**：5 篇（NSFC 系统限制）
- **角色**：**全部一作 / 通讯**——共同一作 / 共同通讯**慎用**（评审会专门核查）
- **质量**：至少 1-2 篇该领域**顶级期刊**或**领域内最高声誉**的工作
- **时效**：3-5 年内的工作占主体（评审重视"近期持续产出"）
- **故事线**：5 篇代表作要能串成"一个学者 5 年聚焦 X 方向的演进故事"——而非"5 个独立的好工作"

**故事线写法（每篇代表作后追加 1-2 句）**：
> "[1]（202X，通讯）首次提出 …… 框架，奠定本项目方向的理论基础。
> [2]（202X，一作）将 [1] 的框架扩展到 …… 场景，验证 …… 命题。
> [3]（202X，通讯）通过 …… 实验进一步验证，被 [国际顶刊] 等同行积极引用 / 跟进。
> [4]（202X，通讯）解决了 [1-3] 中 …… 的方法学瓶颈。
> [5]（202X，通讯）将方法学产业化应用于 …… 场景。
>
> 上述工作构成了本项目"方法学奠基 → 场景验证 → 瓶颈突破 → 应用拓展"的完整链条。"

### 策略 5：现场答辩准备（杰青二面 / 优青部分）

杰青进入会评后会有**现场答辩 (10-15 分钟 PPT + 5-10 分钟问答)**。准备要点：

- **PPT 不超过 15 页**：标题 + 自我介绍（含 5 年方向 + 标志性突破 1 页）+ 立项 / 创新 / 方案各 2-3 页 + 5 年蓝图 1 页
- **首页必须有 1 张"项目命名图"**：把项目级概念 / 假说放在显著位置
- **预演问题**：评审最常问 5 类问题
  1. "你的标志性贡献是什么？" → 提前准备 30 秒电梯演讲
  2. "你的项目和已有 X 工作的关键区别？" → 准备一个对比表
  3. "你 5 年的方向是什么？" → 准备一张方向演进图
  4. "如果方法不奏效，你怎么办？" → 准备 Plan B
  5. "你的团队 / 平台支撑是什么？" → 准备团队结构图

## 三、优青范式（与杰青的差异）

优青是杰青预备队，整体范式与杰青一致，但有几处差异：

| 维度 | 杰青 | 优青 |
| --- | --- | --- |
| **代表作** | 5 篇全 1 作 / 通讯 | 5 篇，可有 1 篇共同 1 作 / 通讯 |
| **标志性突破** | "3 项" | "1-2 项" 即可 |
| **5 年蓝图** | 必须给出"国际重要研究者"目标 | "形成独立研究方向"即可 |
| **答辩** | 杰青二面 | 部分学部不答辩 |
| **现场答辩 PPT 页数** | 15 页 | 10-12 页 |
| **资助强度** | 400 万 / 5 年 | 200 万 / 3 年 |

## 四、杰青 / 优青最常见的 5 类失败原因

### 1. "方向飘移"型

5 年前做 A、3 年前做 B、今年又申请 C——评审看一眼代表作时间线就否决。

**Fix**：在立项依据段 1 用 1 段说"为什么 5 年来都聚焦在 A，本项目是 A 的逻辑延伸"。如果方向真有调整，在"研究基础"中给出"过渡性成果"——B → C 的转向已有 1-2 篇产出。

### 2. "代表作硬度不够"型

5 篇代表作均为 SCI 二区 / 三区，无该领域顶刊—— hardness 不到杰青门槛。

**Fix**：申请前用 3-5 年时间冲 1-2 篇本领域顶刊；不要用"数量"代替"硬度"。

### 3. "项目格局太小"型

研究方案像面上申请——只解一个具体小问题。

**Fix**：杰青范式必须给"5 年蓝图 + 学科地位定位"。本项目要能撑起一个学科方向。

### 4. "无标志性概念"型

创新点是"3 条工作量描述"——评审记不住你这个项目。

**Fix**："3+1 写法"——给项目一个可记忆的标签 / 假说 / 框架名。

### 5. "团队 / 平台薄弱"型

依托单位不是国家重点实验室；团队中无重要合作者；过往未承担过 NSFC 重点 / 重大类项目。

**Fix**：
- 优先依托单位的国家重点实验室申报
- 邀请 1-2 位重量级共同申请人（已合作发表 SCI ≥ 1 篇）
- 完成 1 个面上 / 重点项目作为基础

## 五、注意事项

- 本文件总结的"杰青范式"基于公开发布的杰青获资助专家公开讲座、学校官网公开材料、《国家杰出青年科学基金完整申请书选编》（公开本）等。**不引用任何具体获奖人姓名 / 申请书原文**。
- 杰青评审具有**强烈的学部偏好**：F 学部偏 SOTA + 系统性贡献；G 学部偏理论建构 + 中国情境；H 学部偏机制 + 转化等（详见 `references/reviewer-perspective.md` 学部偏好节）。
- 杰青 / 优青是 NSFC 人才类项目中竞争最激烈的——历年资助率约 5-10%。**写作的最大杠杆不是"写得多漂亮"，而是"5 年聚焦方向 + 1-2 篇标志性论文 + 1 个清晰命名概念"**。
- 最终请以当年 NSFC 官网公布的《国家杰出青年科学基金项目管理办法》《国家优秀青年科学基金项目管理办法》为准。
