# NSFC 评审心理学：函评 / 会评专家的"行为经济学"

`reviewer-perspective.md` 讲的是**制度**（5 维评分、A-E 五档、上会规则）；本文件讲的是**心理学**——同一份申请书放在不同评审手里，为何会产生 A 与 D 的差异？理解这些"非制度变量"，能让你在 5 大模块写作时自觉规避评审的认知偏误，把"灰区"申请推向资助线之上。

> 本文件源于 NSFC 公开制度文件、《中国科学基金》评审综述、获资助 PI 的公开讲座 / 博客分享、以及科研处培训会上 NSFC 科学处主任的部分公开发言整合而成。具体心理倾向年度有差异，**最终以当年评审实践为准**。

---

## 一、函评 5 维的实际权重（不等权）

制度上 5 维并列，但评审专家心理上**并不等权**。多年评审综述显示的"实际权重"约为：

| 维度 | 制度上 | 心理实际权重 | 关键特征 |
| --- | --- | --- | --- |
| 创新性 | 1/5 | ~30% | **D = 直接淘汰**，不可救赎 |
| 科学价值 | 1/5 | ~25% | 与创新性高度相关 |
| 研究方案可行性 | 1/5 | ~20% | D 还有救（"建议补充预实验后再申报"） |
| 申请人基础 | 1/5 | ~20% | 跨界硬伤致命，但渐进迁移可接受 |
| 预算合理性 | 1/5 | ~5% | 几乎不会单独导致退稿，但**异常低预算 = 信号失真** |

**写作启示**：
- 创新性段落要**不惜版面**做漏斗 + 对比表，让 30 秒扫读的专家也能 get；
- 可行性写"差"通常只是"建议补预实验后重新申报"——还有命；可行性写得**像愿望清单**才致命；
- 预算异常低不会加分，反而暴露"申请人不懂自己课题成本"。

## 二、A-E 五档评议的"心理锚定"

评审专家在**第一份**申请书的打分上会消耗最多时间，后续打分会向第一份"锚定"。心理实证统计：

- **A 档**通常占 ≤ 20%（资深专家更克制，只有 10-15%）；
- **B 档** ~40%——"中位"，是评审的安全选择；
- **C 档** ~30%——本质是"客气拒绝"，等价于"我没看出大毛病但也没看到亮点"；
- **D 档** ~10%——明确反对，需理由；
- **E 档** < 5%——程序问题或学术不端嫌疑。

**写作启示**：
- "C 档是客气拒绝"——意味着**让评审找不到亮点 = 死刑**。每个模块必须有 1 个**让人记得住的钩子**（一个数字、一张图、一句反共识陈述）；
- 锚定效应意味着**摘要前 100 字 + 创新点 3 行**决定整体打分基调；后面写得再好也只能在已有锚定上 ±0.5 档微调。

## 三、30 秒决策机制（"摘要-创新点-代表作"三件套）

NSFC 函评每位专家手上 15-25 份本子，要 2 周内审完，**每份平均深读 30-60 分钟**。但**深读之前**有 30 秒的"是否值得深读"决策环节：

1. **0-10 秒**：扫摘要前 100 字（中文 + 英文版各看一眼）；
2. **10-25 秒**：跳到"特色与创新之处"前 3 行；
3. **25-30 秒**：跳到申请人代表作前 3 篇。

这 30 秒决定下面 5 分钟的阅读姿态：
- 三件套都强 → 进入"找亮点"模式（积极偏差）；
- 三件套有 1 件弱 → 进入"找问题"模式（消极偏差）；
- 三件套有 2 件弱 → 直接定 C/D，剩下时间快速浏览找佐证。

**写作启示**：摘要 / 创新点 / 代表作 3 个区域必须**视觉上对齐**——全本子的"门面"。这也是为什么资深 PI 会花申请书写作总时间的 **30%** 在这三个不到 1 页的内容上。

## 四、4A1B vs 5A 的本质差别（会评决战）

很多 PI 误以为"4A1B 和 5A 差别不大，只差一份"。错。**4A1B 在会评中是高风险**，5A 是低风险，原因是：

- **5A**：上会基本只是程序，学科组扫一眼就过；
- **4A1B**：那 1B 的评审意见会**作为讨论起点**——
  - 如果给 B 的评审恰好是学科组成员或他/她的学生在场 → 会评中那条 B 意见被放大成"必须严肃讨论"；
  - 如果给 B 的评审不在场 → 学科组其他成员要么"补刀同意 B"（保险），要么"质疑 B 的合理性"（看代表作 / 看 PI 履历）。

**经验数据**（来自坊间统计，非官方）：
- 5A 上会通过率 ~95%；
- 4A1B 通过率 ~60-70%（看 1B 的具体意见 + 学科组组成）；
- 3A2B 通过率骤降到 ~30%。

**写作启示**：与其追求 5A，不如**消灭可能给 B 的"软肋"**——通常软肋来自方案章节缺技术路线图、或代表作有 1-2 篇"和申请方向看似无关"的论文（其实有关，但没在工作基础里说清楚）。

## 五、学部偏好真实差异（写作时必须切换"语言"）

各学部因学科文化不同，**评审用语和重点完全不同**：

### F 学部（信息科学）— F02 计算机科学

- **代表作权重**：CCF-A 类会议 / 期刊（NeurIPS / ICML / SIGCOMM / OSDI / S&P / TPAMI）；
- **创新性话语**：偏好"算法创新"（新模型 / 新损失函数 / 新理论保证），反感"在 X 数据集上调参 + SOTA"；
- **可行性话语**：要求**开源代码 + 算力声明**（"我们将在 8×A100 上训练 ~120 GPU 天"）；
- **致命伤**：纯刷榜、缺少理论分析、缺少可复现细节。

### G 学部（管理科学）— G02 工商管理

- **代表作权重**：UTD-24（顶级管理类）/ FT-50（金融经济商学院）；中文期刊《管理世界》《经济研究》《管理科学学报》；
- **创新性话语**：偏好"理论贡献 + 中国情境 + 因果识别"；反感"模型黑箱 + 缺机制解释";
- **可行性话语**：要求**数据来源具体**（CSMAR / Wind / 自采访谈样本量）；
- **致命伤**：缺因果识别策略、缺中国情境特殊性、堆砌外文文献而无本土对话。

### H 学部（医学科学）

- **代表作权重**：IF（影响因子）+ **临床转化潜力**；NEJM/Lancet/Cell/Nature Medicine 是天花板；专科顶刊（Circulation / JAMA Oncol / Gut / Hepatology）也很硬；
- **创新性话语**：偏好"机制 + 干预靶点 + 临床可达"三位一体；
- **可行性话语**：要求**伦理批件 / 样本来源 / 动物模型 / 队列规模**全部具体；
- **致命伤**：纯生物学机制无临床落点、缺人类样本、伦理风险写得太轻。

### E 学部（工程与材料）

- **代表作权重**：实物原型 / 工程指标（性能、效率、寿命、成本）；顶刊 Adv Mater / Nat Energy / JACS 等；
- **创新性话语**：偏好"性能突破 + 关键指标领先"；
- **可行性话语**：要求**中试 / 放大可达性**写清楚；
- **致命伤**：纯材料合成无应用场景、性能数据缺对照基线、缺工程化路径。

### B 学部（化学）/ A 学部（数理）

- B：偏好"机理清晰 + 选择性 / 产率 / 立体构型"等具体化学指标；
- A：偏好"问题本身的数学价值 + 方法的可推广性"，对"应用"几乎不感兴趣（数学）；统计 / 概率方向 A011 等会偏重数据驱动方法。

**写作启示**：申请前**精读目标学部近 3 年公布的"评审综述"**（《中国科学基金》刊载），用学部的"母语"写本子。F 学部本子写得像 G 学部腔调（堆 UTD-24）= 立刻被判跨界。

## 六、会评 5-15 分钟陈述 + Q&A（重点 / 重大 / 杰青）

会评答辩是**重点项目以上**人才类项目的"决战"：

- **PPT 总时长**：5 分钟（杰青）/ 8 分钟（优青）/ 10-15 分钟（重点 / 重大）；
- **前 30 秒 = 整场印象**——开场必须用**1 句反共识陈述 + 1 张数据图**抓住学科组注意；
- **核心 1 张图**：用**一张图讲完研究问题 + 创新点 + 已有数据**（即 NIH 所说的 "money figure"）；
- **Q&A 的 3 大杀手问**：
  1. "你这个工作和 [某海外大牛] 去年的 Cell 文章有何区别？"
  2. "如果 Aim 1 失败，Aim 2/3 是否还能做？"
  3. "为什么是你做、不是 [同领域另一个团队] 做？"
- **失分点**：PPT 文字过密、不会"用自己的代表作打补丁"、答辩超时被打断。

**写作启示**：会评 PPT 不是申请书的浓缩版，而是**重写一遍**——以"15 张图讲清整个故事"为目标。

## 七、PI 表现失分点 5 类（隐性扣分）

即使本子写得无懈可击，下列 5 类 PI 历史问题会让评审"心理打折"：

1. **资助史空白 / 多次失败**：连续 3 次面上未中——评审会怀疑"是不是有学术圈知道但本子看不出的问题"；
2. **发表记录与申请方向不符**：代表作主题与本子方向**主题词不同 + 没有合理叙事桥接** → "跨界申请"信号；
3. **合作者全是同单位**：缺少跨学科 / 跨单位合作 → 评审认为"团队封闭、信息单一"；
4. **研究基础与本课题割裂**：工作基础章节列了一堆论文但**没有 1 篇直接对应本课题的预实验或方法基础**；
5. **伦理 / 学术不端历史**：哪怕是 5 年前的撤稿、被处分、未结题项目——会评学科组通常知情，且**这是公开的"一票否决"信号**。

**写作启示**：
- 失分点 1-4 可以靠**叙事修辞**部分弥补（在"工作基础"段写"虽然代表作 X 看似与本课题方向不同，但其方法论 Y 是本课题的核心工具，详见预实验图 N"）；
- 失分点 5 无法弥补——必须如实说明并在科研处帮助下处理。

## 八、写作时的"评审心理"自检清单

写完本子后，**请人帮你做下面 5 件事**（不要自己做，自己有盲点）：

1. ✅ 拿走全部正文，只留摘要 + 创新点 + 代表作三件套，问对方"30 秒后你愿不愿意继续读"；
2. ✅ 把创新点 3 + 1（理论 / 方法 / 应用 / 概念）逐条问"和文献的对照在哪一段"；
3. ✅ 把 5 份"假设是函评的小同行"角色给 5 个不同细分方向的同事，问每人会给 A/B/C/D 哪档以及理由；
4. ✅ 把"假设是会评的学科组成员"角色给 1 个非小同行的本学部资深教授，问他会不会"读不下去"；
5. ✅ 模拟 3 大杀手问（与海外文章区别 / Aim 失败应对 / 为什么是你），录音 5 分钟回答，自检是否流畅。

只有**全部 5 项过关**的本子，才有资格送出去。
