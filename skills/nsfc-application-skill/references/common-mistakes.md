# NSFC 申请书高频被毙原因

本文件列出 NSFC 申请书最常见的 5 大类退稿原因，并给出修复方向。多数失败申请书并非"想法不好"，而是"表述未让小同行看出好"。

## 一、创新性表述失焦（最常见，占退稿 30%+）

**典型症状**：

- 创新点写成"研究内容的复述"
- 用"首次"+"系统性"+"全面"等空洞修饰词
- 把"新数据"或"新案例"当作创新

**示例（劣）**：

> 创新点：本项目首次系统性研究了某某地区的某某现象，构建了完整的理论框架。

**修复**：

- 用 "**区别于已有 ……，本项目 ……**" 句式，必须显式对比
- 创新点 ≤ 3 条；分别对应 思想 / 方法 / 应用
- 每条创新点 1 段（5-8 行）；首句给"总结性结论"

**示例（优）**：

> 创新点 1（思想创新）：区别于既有研究将 X 与 Y 视为单向作用，本项目首次提出 X 与 Y 的双向耦合机制，并以可验证假设 H1 / H2 形式给出。

## 二、关键科学问题宽泛、不可证伪（占退稿 20%+）

**典型症状**：

- 关键科学问题表述为"研究 X 的影响因素"、"探讨 X 的内在规律"
- 关键科学问题与研究内容混为一谈
- 提了 5-6 个问题，每个都泛泛

**修复**：

- 关键科学问题 **2-3 个为宜**，每个独立成段
- 每个问题应满足"**有理论难度 + 可证伪 + 可操作**"
- 用"**在什么条件下，X 是否真的 Y / 为什么 Y / 如何 Y**"句式
- 关键科学问题 ≠ 研究内容；前者是"为什么难"，后者是"我做什么"

## 三、研究方案空洞、缺技术路线图（占退稿 20%+）

**典型症状**：

- 方案章节写成"将采用 XXX 方法 + XXX 数据"，无方法细节
- 没有技术路线图，或图为简单流程框
- 可行性写成"我们有信心、团队经验丰富"

**修复**：

- **必须放 1 张技术路线图**：节点对应研究内容 R1, R2, R3 ...，连线说明依赖关系，输出节点对应预期成果
- 每个研究内容 1-2 段方法细节：含数据 / 模型 / 实验设计 / 显著性判据
- 可行性分析按 4 角度展开：理论可行 + 方法可行（含预实验数据）+ 条件可行（设备）+ 团队可行
- 至少有 1 处"预实验图 / 数据"，作为可行性硬证据

## 四、申请人基础与课题脱节（占退稿 15%+）

**典型症状**：

- 申请人代表作集中在 A 方向，本项目却投 B 方向
- 列出大量与课题无关的 SCI 数量
- "研究基础"章节像"个人简历"

**修复**：

- 代表作精选 5-10 篇，**全部与课题相关**
- 每篇用 1-2 句话说"做了什么 + 与本项目研究内容 R? 的关系"
- 若申请人确有方向调整，需用 1 段说"为什么本项目是顺势延伸"，并给出过渡性成果
- 团队成员各自分工要清晰，不写"协助申请人完成所有内容"

## 五、预算与研究内容不匹配（占退稿 10%+）

**典型症状**：

- 预算明显超标（设备费占 50%+）
- 预算明显压低（材料费 1 万、劳务费 0）
- 国合项目无国际合作交流费
- 研究内容里说"开展 5 次田野调查"，差旅费却不够

**修复**：

- 写完研究方案，**反向倒推预算**：每项研究内容能落到具体的材料 / 测试 / 出差 / 学生工资
- 设备费控制在 ≤ 30%（青年项目 ≤ 5%）
- 劳务费按"研究生数 × 月津贴 × 月数"估算
- 包干制项目仍要在"经费说明"中给出大致结构

## 六、其他常见小毛病

- **摘要不点题**：摘要前 2 句没说"做什么 + 为什么有意义"
- **图表过多 / 过少**：图过多看着花、过少看着空。**建议 4-8 张**
- **格式不规范**：未使用 NSFC 模板；字号 / 行距 / 标题层级随意改动
- **代码错配**：选错二级代码 → 函评专家不是小同行
- **未声明限项**：在研项目数 + 申请数超出当年限项规定
- **学风诚信问题**：抄袭、重复申请、未结题历史项目

## 七、自审清单（写完后逐项核对）

- [ ] 摘要 200-400 字，前 2 句答出"问题 + 意义"
- [ ] 立项依据有文献现状的批判性总结
- [ ] 关键科学问题 2-3 个，每个可证伪
- [ ] 研究内容 R1-R5 编号清晰
- [ ] 技术路线图 1 张，节点 = 研究内容
- [ ] 可行性分析含预实验数据 / 图
- [ ] 创新点 ≤ 3 条，每条与已有工作显式对比
- [ ] 申请人代表作 5-10 篇，全部与课题相关
- [ ] 预算与研究内容互相印证
- [ ] 申请代码（二级 / 三级）已用当年代码表核对
- [ ] 限项规定已核对
- [ ] 已让 1-2 位小同行预审

## 注意事项

- 本文件总结的高频原因来自 NSFC 公开公布的"申请书撰写要点"和大量公开评审反馈分享，比例为经验性估计，不代表官方统计。
- 各学部 / 学科退稿原因分布略有差异，工科偏可行性、理科偏创新性、人文社科偏问题清晰度。
