# 工作基础与条件撰写 Prompt

> 用途：起草 / 重写 NSFC 申请书"研究基础与工作条件"模块（含 4 个子节）。

## 模板

```
你是 NSFC 申请书写作助手。请帮我起草「研究基础与工作条件」章节。

【输入信息】
- 申请人姓名 / 职称 / 研究方向：{{……}}
- 申请人近 5 年代表作（5-10 篇，标注一作 / 通讯）：{{论文 1：作者，期刊，年份，主题；……}}
- 申请人正在承担的科研项目（含 NSFC 与其他）：{{项目 1：项目名，编号，资助机构，金额，年限，是否结题；……}}
- 申请人已结题 NSFC 项目（限近 5 年）：{{项目 1：编号，名称，结题情况，主要产出；……}}
- 团队成员（共同申请人）：{{成员 1：姓名，职称，研究方向，与本项目分工}}
- 实验室 / 平台 / 仪器：{{……}}
- 协作单位：{{……}}

【写作要求 — 4 个子节】

## 1. 研究基础（约 1000-1500 字）
- 围绕本项目的「研究内容 R1-Rn」组织代表作，**而非按时间或期刊层次**
- 每篇代表作 1-2 句话：(a) 做了什么；(b) 与本项目研究内容 R{{i}} 的关系
- 末段总结 1 段："上述工作奠定了本项目 …… 的基础"
- 不要列出与课题无关的 SCI 数量

## 2. 工作条件（约 400-600 字）
- 实验室：列出国家 / 教育部 / 省部级重点实验室全名
- 仪器：仅列与本项目相关的大型仪器（设备型号 + 主要性能 + 可使用机时）
- 计算资源（如适用）：服务器规模 / 节点数
- 协作单位：列出 1-3 个，并说明合作内容（已有合作论文 / 共建项目）

## 3. 正在承担的与本项目相关的科研项目情况（约 300-500 字）
- 必须**全部列出**正在承担的科研项目（不限 NSFC，含省部级 / 单位 / 横向）
- 对每个项目说明 "**与本项目的区别和联系**"，避免被扣"重复资助"分
- 标准句式："项目 X 聚焦 ……，本项目延伸 / 区别于 …… 之处在于 ……"

## 4. 完成国家自然科学基金项目情况（限近 5 年，约 300-500 字）
- 仅列近 5 年已结题的 NSFC 项目（含青年 / 面上 / 国合等）
- 每个项目 1 段：项目编号 + 名称 + 结题情况 + 主要产出（论文 / 专利 / 数据库）+ 评价等级（如有）
- 若无 — 写"申请人此前未承担 NSFC 项目"
- 若有未结题或结题差 — 必须主动说明改进措施

【自检清单】
- [ ] 代表作是否围绕研究内容组织（不是按期刊层次或时间）？
- [ ] 每篇代表作是否说明与 R{{i}} 的对应关系？
- [ ] 是否列出全部在研项目（含非 NSFC）？
- [ ] 是否说明在研项目与本项目的区别？
- [ ] 是否列出近 5 年所有已结题 NSFC 项目？
- [ ] 团队成员分工是否清晰（不写"协助申请人"）？

请先输出 4 个子节的"框架"（每节 1-2 句话），等我确认后展开。
```

## 团队成员分工写法

```
- 申请人 {{姓名}}：负责总体设计、关键科学问题 KP1 的理论建模、R1 R3 的实验设计与论文撰写
- 共同申请人 {{姓名}}（{{职称}}）：负责 R2 的算法开发与代码实现、KP2 的方法论支撑
- 共同申请人 {{姓名}}（{{职称}}）：负责 R4 的数据采集、数据库建设、与协作单位 {{……}} 的对接
- 博士研究生 N 人：参与 R2 R3 R4 的具体实施
- 硕士研究生 N 人：参与数据预处理、文献整理
```

## 代表作引用格式（建议）

每条按以下格式列出（中文文献按 GB/T 7714，英文文献按通用格式）：

```
[1] 作者. 论文题目[J]. 期刊, 年份, 卷(期): 页码. (本人 一作 / 通讯; SSCI/SCI 一区/二区)
    与本项目关系：本文提出的 …… 方法将作为 R2 的方法基础。
```

## 注意事项

- 在研项目"必须全部列出"是 NSFC 明确要求；漏列将被认定为"科研诚信问题"
- 团队规模建议：青年项目 3-5 人；面上 5-8 人；重点 / 杰青 8-15 人；过大反而显得"挂名"
- 工作条件不要"凡是单位有的都写" — 只写与本项目直接相关的
