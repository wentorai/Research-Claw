# 研究内容撰写 Prompt

> 用途：根据立项依据，起草 / 重写 NSFC 申请书"研究内容、研究目标、拟解决的关键科学问题"模块（约 1500-2500 字）。

## 模板

```
你是 NSFC 申请书写作助手。请帮我起草「研究内容、研究目标与拟解决的关键科学问题」章节。

【项目基本信息】
- 项目类别：{{面上 / 青年 / 重点 / 杰青 / …}}
- 研究主题：{{一句话主题}}
- 立项依据已写出的 3 个流派 / 局限简述：{{摘抄前章 5 段总结}}

【写作要求】

## 1. 研究目标（3-5 句话）
- 必须有边界：清楚说出"研究什么、不研究什么"
- 必须可检验：每个目标可以对应到一个可观测的判据
- 不要堆形容词："全面""系统""国际领先"等避免

## 2. 研究内容（编号 R1, R2, R3, ...，3-5 个）
- 每个内容点 2-3 段
- 每段结构：(a) 研究问题；(b) 拟采用方法 / 数据 / 模型；(c) 预期产出
- 内容点之间有逻辑递进或并列关系（不可彼此独立无关）

## 3. 拟解决的关键科学问题（2-3 个，必填）
- 每个问题独立成段（200-400 字）
- 每段结构：(a) 问题陈述（句式："在 X 条件下，Y 是否真的 …？/ 为何 …？/ 如何 …？"）；(b) 为什么这是关键科学问题（理论难点 / 已有研究的方法论困境）；(c) 本项目如何切入
- **关键科学问题 ≠ 研究内容**：前者是"理论难点"，后者是"研究任务"

【输入信息】
- 候选研究问题（2-3 个）：{{用户输入}}
- 候选方法 / 数据：{{用户输入}}
- 团队已有的相关方法储备：{{用户输入}}

【自检清单】
- [ ] 研究目标是否限定边界且可检验？
- [ ] 研究内容编号 R1-R5 是否清晰？
- [ ] 研究内容与研究目标是否逐条对应？
- [ ] 关键科学问题数量在 2-3 个？
- [ ] 每个关键科学问题是否可证伪？
- [ ] 是否避免把研究内容复述为关键科学问题？

请先输出 1 段"框架性回答"（研究目标 1 句 + R1-Rn 标题 + 关键科学问题标题），等我确认后展开全文。
```

## 句式工具箱

- **关键科学问题陈述句式**：
  - "在 {{条件 X}} 下，{{现象 Y}} 是否仍然 {{表现 Z}}？"
  - "{{机制 A}} 与 {{机制 B}} 在 {{情境 C}} 中的相对贡献是什么？"
  - "如何通过 {{方法 D}} 识别 {{效应 E}} 的因果路径？"
- **研究内容点段首句**：
  - "R{{n}}：{{内容标题}} — 本部分聚焦 {{问题}}，拟采用 {{方法}}，回答 {{对应关键科学问题}}。"

## 注意事项

- 研究内容的"颗粒度"：5 个内容点比 3 个更稳妥（评审认为后者"信息量不足"），但超过 7 个会显得"摊大饼"
- 如果是青年项目，内容控制在 3 个左右，深做不广做
- 关键科学问题切忌写成"研究 X 与 Y 的关系" — 这不是科学问题，是研究主题
