# 立项依据撰写 Prompt

> 用途：根据用户提供的研究问题与背景信息，起草 / 重写 NSFC 申请书"立项依据"模块（约 3000-5000 字）。

## 模板

```
你是一位资深 NSFC 评审专家与申请书写作助手。请帮我起草 NSFC「{{项目类别}}」项目申请书的「立项依据」章节。

【项目基本信息】
- 项目类别：{{面上 / 青年 / 重点 / 杰青 / 优青 / 国合 / 重大研究计划 / 其他}}
- 拟申请代码：{{学部代码，如 G0301}}
- 研究主题：{{一句话研究主题}}
- 关键词：{{5 个关键词}}
- 申请人方向：{{申请人近 5 年聚焦方向，1-2 句}}

【已有信息】
- 国内外代表性流派 / 方法（≥ 3 个）：{{流派 1：…；流派 2：…；流派 3：…}}
- 各流派的局限：{{流派 1 局限：…；流派 2 局限：…；流派 3 局限：…}}
- 本项目拟提出的研究问题：{{2-3 个候选研究问题}}
- 已有的预实验 / 前期成果（如有）：{{……}}

【写作要求】
1. 章节字数：3000-5000 字（青年项目 2000-3000 字）
2. 结构按以下 4 段组织（不要按时间编年）：
   - 第 1 段（约 400-600 字）：研究意义 — 学术价值 + 应用前景，前两句必须点题
   - 第 2-4 段（每段 600-1000 字）：国内外研究现状 — 按"代表性流派 / 方法"组织，每段先客观描述再批判性指出局限
   - 第 5 段（约 400-600 字）：批判性总结 + 自然引出本项目研究问题
3. 文风：克制学术，避免"非常""极其"等程度副词；多用"显著""稳健""异质性"等术语
4. 引用：在文中以 "（作者，年份）" 或 "[1]" 风格留出引用占位，方便后续插入参考文献
5. 关键提示：必须包含 ≥ 1 处"批判性总结"句式，例如"已有研究虽然 ……，但尚未 ……"

【自检清单（请你写完后自查并报告）】
- [ ] 第 1 段前两句是否答出"做什么 + 为什么重要"？
- [ ] 是否按"流派 / 方法"组织而非时间编年？
- [ ] 是否对每个流派都给出了批判性局限？
- [ ] 研究问题是否在第 5 段自然涌现而非突兀给出？
- [ ] 是否避免使用了"首次""填补空白""国际领先"等空泛词？
- [ ] 总字数是否在目标区间内？

请先输出"写作大纲"（5 段每段 1-2 句话），等我确认后再展开成全文。
```

## 进阶选项

- 如果项目类别是 **杰青 / 优青**，将"研究意义"改写为"申请人 5 年聚焦方向 + 本项目作为方向延伸"
- 如果项目类别是 **重大研究计划**，第 1 段必须明确对应到当年指南方向
- 如果项目类别是 **国际合作 (NSFC-XXX)**，需要在第 5 段加 1 段说明"中外合作必要性"

## 注意事项

- 请勿在 Prompt 中要求 Claude 编造具体文献引用 — 文献由用户后续补充
- 涉及具体数据、政策、机构名称时，建议用户事后核对
