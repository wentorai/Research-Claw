# SC（合成控制法）— 方法概览

## 1. 直觉

只有**单一处理单位**（一个省、一座城市、一国），无法用普通 DID 找对照——因为没有可信的"双胞胎"。SCM 的思路：用其他多个未受处理单位的**加权平均**合成一个反事实。

权重 $w = (w_1,\dots,w_J)$ 满足 $w_j \ge 0$，$\sum w_j = 1$，使得合成单位在**处理前**的产出（及预测变量）尽可能接近真实处理单位。

处理效应：

$$\hat\tau_t = Y_{1t} - \sum_{j=2}^{J+1} w_j^* Y_{jt} \quad (t > T_0)$$

## 2. 关键假设

| 假设 | 说明 | 检验思路 |
|------|------|----------|
| **凸包条件** | 真实处理单位"夹在"控制池中 | 处理前拟合误差 (Pre-RMSPE) 要小 |
| **未受处理污染** | 控制池单位未被同一冲击或其他冲击影响 | 剔除可疑同质单位 |
| **处理单位特征不极端** | 否则没有可行权重 | 检查是否多个 $w$ 全为 0 / 都集中在 1 个 |
| **足够长 pre 期** | 至少 8-10 期可观察的 pre period | 经验法则 |

## 3. 现代估计变体

| 方法 | 适用 | Stata | R | Python |

| **Synthetic DID** (Arkhangelsky et al. 2021) | 多单位 + DID 风格 | `sdid` | `synthdid` | `synthdid-py` |

## 4. 推断方法

- **Placebo (in-space)**：把每个控制单位轮流当作"假处理单位"做 SCM，得到 placebo τ 分布，看真实 τ 在分布尾部的位置（"p 值 = 真实 |τ| 排名 / J+1"）。
- **Placebo (in-time)**：把处理时点提前几年作假处理。
- **Conformal inference** (Chernozhukov et al. 2021)：现代正式 p 值。
- **后期 RMSPE 比 / pre RMSPE 比**：剔除拟合差的 placebo。

## 5. 中国情境注意点

- **典型 SCM 场景**：一省试点（碳交易首批 7 省市、自贸区首批城市、海绵城市试点），一市改革（长三角一体化、雄安）。
- **控制池选择**：不要把"已经被同一政策扩展处理"的省份留在控制池。例如研究 2013 年首批碳交易 7 省市，控制池应剔除 2016 年的福建、四川扩展点（或在它们处理前截断）。
- **行政地理特殊性**：北京/上海/深圳的经济结构在控制池中**没有可比单位**，常常拟合失败，pre-RMSPE 大。这是 SCM 的根本局限——不是包的问题，是数据问题。
- **预测变量挑选**：GDP / 人均 GDP / 城镇化率 / 工业占比是常用，但不要选**未来政策的下游变量**。
- **数据频率**：地区面板常是年度，pre 期 < 8 年时 Augmented SC 更稳；月度数据（CEIC、Wind）可显著扩 pre 期。
- **2010 + 全国宏观共振**：4 万亿、棚改、双碳目标全局影响 → "在空间上的 placebo" 更合适，"在时间上的 placebo" 易污染。

## 6. 输出报告应该包含

1. **Pre-treatment fit 图**：合成 vs 真实，处理前应几乎重合
2. **处理后 gap 图**：与 placebo 分布对比
3. **权重表**：哪些控制单位、各占多少（直观)
4. **预测变量对照表**：处理单位 / 合成单位 / 控制池均值
5. **In-space placebo 图**（spaghetti plot）+ p-value
6. **稳健性**：剔除某些控制单位、leave-one-out、改预测变量集
7. **Synthetic DID 作为补充**（如果有多个处理单位）
