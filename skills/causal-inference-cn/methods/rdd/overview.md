# RDD（断点回归）— 方法概览

## 1. 直觉

存在一个连续 running variable $X$（"分数线"、"年龄线"），处理 $D$ 在阈值 $c$ 处出现不连续：

- **Sharp RDD**：$D = \mathbf{1}[X \ge c]$（阈值附近完全决定处理）
- **Fuzzy RDD**：阈值导致处理概率跳跃但不到 1（用 $\mathbf{1}[X \ge c]$ 作为 IV）
- **Kink RDD**：处理强度的斜率在 $c$ 处出现拐点

阈值附近"刚好过线"和"刚好没过线"的个体在协变量上近似随机 → 局部 ATE 可识别。

主参数（sharp）：

$$\tau_{SRD} = \lim_{x \downarrow c} E[Y|X=x] - \lim_{x \uparrow c} E[Y|X=x]$$

## 2. 关键假设

| 假设 | 说明 | 检验思路 |
|------|------|----------|
| **连续性** | 反事实 $E[Y(0)|X]$ 在 $c$ 处连续 | 协变量在 $c$ 处不应跳跃 |
| **个体不能精确操纵 X** | 否则刚好过线的个体被自我选择 | McCrary / `rddensity` 密度断点检验 |
| **唯一断点效应** | $c$ 处不应同时存在其他政策跳跃 | 制度背景核查 |

## 3. 现代估计标准（Calonico-Cattaneo-Titiunik 2014）

- 局部多项式估计（一般用 1 阶或 2 阶）
- **MSE-optimal 带宽** + **bias-corrected**（CCT 的 `rdrobust`）
- **Robust 95% CI**（不要直接报 conventional CI）

公式（一阶局部线性）：

$$\hat\tau = \hat\alpha_+ - \hat\alpha_-$$

其中 $\hat\alpha_\pm$ 来自在 $[c, c+h]$ 与 $[c-h, c]$ 内的局部加权回归（三角核）。

## 4. 中国情境注意点

- **行政线（行政区划/户籍/学籍）**：常常是天然 RDD 来源（高考分数线、土地审批面积阈值、企业规模阈值）。但**很多阈值自带其他政策捆绑**（如规模以上工业企业 2000 万元营收阈值同时触发统计上报、税收审查、环保监管），单一断点 → 多重处理。
- **操纵问题在中国常很严重**：
  - 高考分数：扩招、加分政策导致刚好过线的不是真"过线"
  - 企业规模：营业额"凑数"过 2000 万阈值（"规上"操纵）
  - 必须做 `rddensity` McCrary 检验，**不通过就不要用 RDD**
- **数据精度**：CSMAR 营业额单位可能是万元，CHFS 收入单位是元，断点位置必须按数据本身单位写。
- **bunching 现象**（聚集在某税档）：常见于个税起征点、增值税小规模阈值；这本身是 RDD 不能用的信号，但反过来可作为 bunching estimator。
- **样本量限制**：阈值附近样本量小 + 中国微观调查样本本就稀疏 → **不要把带宽缩到 50 个观测以下**，否则估计不稳。

## 5. 输出报告应该包含

1. **RDD plot**：横轴 X，纵轴 Y bin 平均，叠加局部多项式拟合线（左/右）
2. **主估计**：`rdrobust` 输出 conventional / bias-corrected / robust 三列
3. **密度检验**：`rddensity` McCrary 图与 p 值
4. **协变量平衡**：在 $c$ 处对每个协变量做同样 RDD（应不显著）
5. **带宽稳健性**：0.5h, h, 1.5h, 2h 多个带宽下的系数曲线
6. **多项式阶数稳健**：p=1, 2, 3 的对照（避免 Gelman-Imbens 警告：高阶多项式不稳）
7. **donut hole 检验**：剔除阈值附近 ε 的样本，看主结果是否稳定
