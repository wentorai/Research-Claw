# IV（工具变量）— 方法概览

## 1. 直觉

当回归子 $D$ 与误差项相关（内生性：遗漏变量、反向因果、测量误差），OLS 不一致。找一个变量 $Z$（工具变量）满足：

- **相关性 (Relevance)**：$\text{Cov}(Z, D) \neq 0$
- **外生性 / 排他性 (Exclusion)**：$\text{Cov}(Z, \varepsilon) = 0$（$Z$ 仅通过 $D$ 影响 $Y$）

二阶段最小二乘 (2SLS)：

第一阶段：$\hat D_i = \pi_0 + \pi_1 Z_i + X_i'\gamma + v_i$
第二阶段：$Y_i = \beta_0 + \beta_1 \hat D_i + X_i'\delta + u_i$

LATE 解读 (Imbens-Angrist 1994)：当 $Z$ 二元、单调，$\hat\beta_{IV}$ 估计的是 **complier**（被 $Z$ 推动改变 $D$ 的子总体）的处理效应，**不是 ATE**。

## 2. 关键假设

| 假设 | 说明 | 检验思路 |
|------|------|----------|
| **相关性** | 第一阶段非弱 | F 统计量；Stock-Yogo / Olea-Pflueger |
| **排他性** | $Z$ 不通过其他渠道影响 $Y$ | 不可直接检验，需理论论证 + Hansen J（仅过度识别时） |
| **单调性** | 没有 defier（被 $Z$ 推向反方向的人） | 制度逻辑论证 |
| **SUTVA** | 个体处理彼此独立 | 同 DID |

## 3. 弱工具的危险（Stock-Yogo / Olea-Pflueger）

经验法则：第一阶段 F < 10 → 弱工具，2SLS 偏向 OLS、CI 严重失真。

更严谨：**Olea-Pflueger (2013) effective F**，阈值因 worst-case bias 容忍度而异（通常 23.1 对应 5% 的 worst-case bias）。

## 4. 过度识别 (Over-identification, Hansen J)

工具数 > 内生变量数时，可检验所有工具是否同时外生。但通过 J 检验**不等于**外生性成立——只是工具间相互一致。

## 5. 中国情境注意点

- **历史/地理工具的滥用**：
  - "明清进士数" → 现代经济变量
  - "省内河流密度" → 现代基础设施
  - 这类工具被审稿人质疑成"什么都能搭"。**必须给出排他性的具体故事**：为什么工具**只通过**目标渠道影响 Y。
- **"Bartik / shift-share IV"**：行业增长 × 历史份额。在中国研究中常用于汇率冲击、贸易冲击（如中国加入 WTO 的县级出口暴露）。注意 Goldsmith-Pinkham-Sorkin-Swift (2020) 的批评：本质识别假设是 **历史份额** 外生。
- **同一区域多 IV 重复使用**：同一地理工具在多篇论文中已用尽，新研究若仍依赖，需要新维度（时变、按行业的份额）。
- **政策"准自然实验"作 IV**：常用，但容易触发同一政策被多论文使用、且与其他时间 trend 混淆。
- **数据精度**：CSMAR 的省份代码、行业代码不同时点编码不同（GB/T 4754-2002 vs 2011 vs 2017），合并工具变量时务必确认编码一致。

## 6. 输出报告应该包含

1. OLS 与 2SLS 系数对比（说明内生性方向）
2. **第一阶段回归**（必报）：$Z$ 系数 + F 统计量
3. **Olea-Pflueger 有效 F**（>23 较安心）
4. **Hansen J**（多工具时）
5. 排他性的**理论论证段落**（这部分是审稿人最关注）
6. 安慰剂（用一个理论上不应通过 $D$ 影响 $Y$ 的"假"结果变量做同样回归，应不显著）
7. **简化型 (Reduced Form)**：$Z$ 直接对 $Y$ 回归——若 RF 不显著，IV 显著就要怀疑了
