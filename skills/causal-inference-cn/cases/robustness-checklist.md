# 稳健性检验清单

任何因果识别论文应至少跑 5 项稳健性，分类如下。

## 跨方法通用

- [ ] **聚类稳健 SE**：处理在何层级聚类至少在该层级。少于 30 个 cluster → wild cluster bootstrap (`boottest`)
- [ ] **不同样本期**：剔除疫情年（2020-2022）、剔除危机年（2008、2015 股灾）
- [ ] **不同样本范围**：剔除直辖市、剔除某行业（金融、地产）、仅 A 股
- [ ] **缩尾处理**：1% 与 5% 双对照
- [ ] **冬青效应 (winner's curse)**：是否过度依赖少数大权重观测（leverage plot）
- [ ] **替代因变量度量**：lnY vs ln(1+Y)、原值 vs 归一化、t 期 vs t+1 期

## DID 专属

- [ ] **平行趋势 F 检验**（事件研究图）
- [ ] **安慰剂时间**：处理时点提前 2/3 年
- [ ] **安慰剂处理组**：随机抽取处理 unit, 1000 次
- [ ] **不同估计量交叉**：TWFE / csdid / did_imputation / sun_abraham
- [ ] **样本动态**：仅平衡面板 vs 不平衡面板
- [ ] **延迟效应**：把窗口扩到 t+10，看是否仍显著
- [ ] **空间溢出**：剔除处理单位的相邻 unit 作对照

## RDD 专属

- [ ] **多带宽**：0.5h, h, 1.5h, 2h
- [ ] **多多项式阶**：p = 1, 2, 3
- [ ] **多 kernel**：triangular, epanechnikov, uniform
- [ ] **McCrary 密度检验**
- [ ] **协变量平衡**：每个协变量做同样 RDD
- [ ] **Donut hole**：剔除阈值 ±ε 样本
- [ ] **Placebo cutoff**：在非阈值点做 RDD，应不显著

## IV 专属

- [ ] **第一阶段 F**（KP-rk Wald F 异方差稳健）
- [ ] **Olea-Pflueger effective F**（>23.1 较安心）
- [ ] **Hansen J**（多工具）
- [ ] **简化型 (RF)**：Z 直接对 Y，应显著
- [ ] **LIML / Fuller** 作弱工具稳健性
- [ ] **Anderson-Rubin CI**（弱 IV 也有效）
- [ ] **安慰剂结果变量**：理论上不应被影响的 Y
- [ ] **加入工具的可疑替代渠道**：直接控制 Z 的一些"中间变量"（部分 over-control，但作为表底稳健性）

## SC 专属

- [ ] **In-space placebo**：每个控制单位轮流当假处理
- [ ] **In-time placebo**：把处理时点提前 5 年
- [ ] **Leave-one-out**：剔除最大权重单位
- [ ] **改预测变量集**：去掉/添加 1-2 个变量
- [ ] **Pre-RMSPE 报告**：必须 < 0.10
- [ ] **Synthetic DID** 作补充估计
- [ ] **Augmented SC** (Ridge / GSYN) 对照

## PSM / DR 专属

- [ ] **平衡前后表**（SMD < 0.1）
- [ ] **Love plot**
- [ ] **倾向分密度图**（before / after）
- [ ] **共同支撑剔除比例 < 5%**
- [ ] **多种匹配方法 ATT**：1:1, 1:4, kernel, Mahalanobis, CEM
- [ ] **Entropy balancing** 作对照
- [ ] **DR / DoubleML** 作回应 King-Nielsen 批评
- [ ] **Rosenbaum bounds**：报告 Γ 临界值
- [ ] **倾向分模型设定**：含交互项与多项式

## 中国情境额外检查

- [ ] **行政区划稳定性**：研究期内是否有县改区、撤并；如有，固定在某一年区划码
- [ ] **行业代码版本**：GB/T 4754-2002 / 2011 / 2017 是否一致
- [ ] **数据库快照日**：Wind / CSMAR 导出日期与版本
- [ ] **政策叠加**：研究期内是否有其他全国性政策（双碳、限电、扶贫攻坚收官、疫情）干扰
- [ ] **样本覆盖**：CSMAR 仅上市，结论是否外推；CFPS 仅住户，企业层面看不到
- [ ] **缩尾报告**：上市公司财务连续变量是否 winsor 到 [1%, 99%]
- [ ] **直辖市处理**：北京/上海/天津/重庆是否单列或剔除（行政地位特殊）

## 机制检验（不是稳健性，但同等重要）

- [ ] **中介效应**：Y = α + β D + γ M + ... 且 M = δ D + ... ；现代做法用 `medflex` (R) 而非 Baron-Kenny
- [ ] **调节效应**：异质性分组 + 系数图
- [ ] **机制变量先于结果**：在时间上 t-1 期机制变量 → t 期结果
