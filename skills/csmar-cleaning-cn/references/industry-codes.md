# CSRC 行业代码

中国证监会 (CSRC) 发布的上市公司行业分类是 A 股研究的事实标准。共有两个版本：

- **CSRC 2001**（旧版）— 2001–2012 年使用
- **CSRC 2012**（现行）— 2012 年至今

在 CSMAR 中，`EN_EnterpriseInfo.IndCd` 字段默认存最新分类（CSRC 2012）；部分老论文的旧数据集可能仍带 CSRC 2001 编码。本文件给出两版的对照与映射。

---

## CSRC 2012 行业大类（现行）

字母为大类（A–S），后接 2 位数字为中类。

| 大类 | 名称 | 实证常见处理 |
|---|---|---|
| A | 农、林、牧、渔业 | 保留 |

| E | 建筑业 | 保留 |

| 大类 | 名称 | 与 2012 大致对应 |

| A | 农、林、牧、渔业 | A |
| B | 采掘业 | B |
| C | 制造业 | C |
| D | 电力、煤气及水的生产和供应业 | D |
| E | 建筑业 | E |
| F | 交通运输、仓储业 | G（部分） |
| G | 信息技术业 | I |
| H | 批发和零售贸易 | F |
| **I** | **金融、保险业** | **J（金融）** |
| J | 房地产业 | K |
| K | 社会服务业 | L/N/O（拆分） |
| L | 传播与文化产业 | R |
| M | 综合类 | S |

> ⚠️ **最易踩坑**：CSRC 2001 中 **金融业代码是 `I`**；CSRC 2012 中 **金融业代码是 `J`**。在跨版本数据合并时，"剔除金融业"的代码必须**两个版本都剔**：

```stata
drop if substr(IndCd_2001, 1, 1) == "I" | substr(IndCd_2012, 1, 1) == "J"
```

## 行业代码字段在不同 CSMAR 表中的命名

| 表 | 字段 | 版本 |
|---|---|---|
| `EN_EnterpriseInfo` | `IndCd` / `IndNme` | 默认 CSRC 2012 |
| `STK_LISTEDCOINFOANL` | `IndCd` | 含历史，按年存（同一 Stkcd 可能跨版本） |
| 部分自定义研究表 | `Indus_Code_C` (CSRC) / `Indus_Code_S` (SIC) | 按命名前缀区分 |

> **提醒**：CSMAR 提供多套行业分类——CSRC、申万、Wind、GICS 等。**论文一般用 CSRC**；对照分类（申万一级行业）有时用于稳健性检验。

## 实操：行业固定效应该用哪个粒度？

| 粒度 | 字段示例 | 适用 |
|---|---|---|
| 大类 (一位字母) | `A, B, C, ..., S` | 样本量小、跨行业研究 |
| 中类 (字母+两位数字) | `C13, C14, ...` | **默认**，多数论文使用 |
| 申万一级 | 28 个一级 | 与 CSRC 中类粒度类似，可作稳健性 |
| 申万二级 / 三级 | 更细 | 行业内异质性研究 |

**Stata 实现**：
```stata
* 大类固定效应
egen ind1 = group(substr(IndCd, 1, 1))
xtreg y x i.ind1 i.year, fe

* 中类固定效应（推荐默认）
encode IndCd, gen(ind2)
xtreg y x i.ind2 i.year, fe

* 用 reghdfe（更快，吸收高维 FE）
reghdfe y x, absorb(IndCd year) cluster(Stkcd)
```

## 关于"行业-年"双向固定效应

`absorb(IndCd#year)` 吸收行业-年交互固定效应，等价于"控制行业层面的时变冲击"。**财会顶刊论文常用**，可同时控制金融危机、行业政策冲击、行业景气度等不可观测因素。

```stata
reghdfe y x, absorb(IndCd#year Stkcd) cluster(Stkcd)
```

> 注意：用 `IndCd#year` 时若行业-年 cell 太小（< 20 观测），估计噪声大，可考虑回到大类。
