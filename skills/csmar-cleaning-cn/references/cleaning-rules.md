# 通用清洗规则

中国 A 股实证研究中**约定俗成**的样本筛选规则。本文件给出每条规则的含义、判定字段、以及不同学派的差异。

---

## 规则总览（建议按此顺序执行）

| # | 规则 | 默认做法 |
|---|------|---------|
| 1 | 删除金融业 | 剔除 |
| 2 | 删除房地产业 | 视情况（K 业；保守剔，否则保留） |
| 3 | 删除 ST / *ST / PT | 剔除 |
| 4 | 删除 IPO 当年观测 | 剔除（Trddt - ListDt < 1 year） |
| 5 | 删除 B 股 / H 股双重上市 | 视研究设计 |
| 6 | 关键变量缺失 | 删除（不插补） |
| 7 | 极值缩尾 | 1% / 99% 缩尾 |
| 8 | 样本期 | 通常 1998 之后；2007 后会计准则更稳定 |

---

## 规则 1：剔除金融业

**理由**：银行、证券、保险公司的资产负债表结构与一般工商业公司**根本不可比**（杠杆率、资产周转率等指标失去经济含义）。

**判定字段**：`EN_EnterpriseInfo.IndCd` 或 `STK_LISTEDCOINFOANL.IndCd`

**CSRC 2012 行业代码下**：
- `J` — 金融业（含银行、证券、保险、其他金融）

**CSRC 2001 行业代码下**（2012 之前的论文仍可见到）：
- `I` — 金融、保险业

**Stata 实现**：
```stata
drop if substr(IndCd, 1, 1) == "J"
```

**Python 实现**：
```python
df = df[~df["IndCd"].str.startswith("J")]
```

---

## 规则 2：是否剔除房地产业（争议）

CSRC 2012 行业 `K` = 房地产业。

- **保守派**（公司金融研究主流）：剔除。理由是房地产公司资产估值方式特殊（存货=未售房屋）、杠杆机制特殊。
- **保留派**（实体经济相关研究）：保留。理由是房地产是 GDP 重要组成。

**论文写法**：在样本筛选段落明示"是否剔除房地产业"+ 在稳健性中报告另一种处理的结果。

---

## 规则 3：剔除 ST / *ST / PT

**ST（Special Treatment）** 是中国 A 股特有的"特别处理"标记，发给经营异常的公司：连续两年亏损 → ST → 第三年仍亏损 → *ST → 退市风险。

**理由**：
- 财务数据不正常（已陷入操控盈余 / 亏损）
- 股价受政策性因素（保壳重组等）干扰大
- 退市风险样本与正常样本不可比

**判定方式 A：通过 `Trdsta`**（`TRD_Dalyr` / `TRD_Mnth`）

| Trdsta | 含义 |
|---|---|
| 1 | 正常 |
| 2 | ST |
| 3 | *ST |
| ... | （具体编码以最新数据手册为准） |

```stata
drop if Trdsta != 1
```

**判定方式 B：通过股票简称匹配**

```stata
drop if strpos(Stknme, "ST") > 0 | strpos(Stknme, "PT") > 0
```

> 简称匹配会误剔除如"星途科技"等含 "ST" 字符串的非特别处理公司。**优先用 `Trdsta`**。

---

## 规则 4：剔除 IPO 当年观测

**理由**：
- IPO 当年财务数据有"包装"嫌疑（招股说明书后第一年业绩往往异常高）
- IPO 当年股价波动巨大，事件研究偏差大
- 净资产收益率等比率因募集资金到位时点而失真

**默认做法**：删除 `Trddt - ListDt < 365 天` 的所有观测；更保守做法是删除 IPO 当年 + 次年。

**Stata 实现**：
```stata
* 假设 ListDt 已 merge 进来
drop if (Trddt - ListDt) < 365
```

**Python 实现**：
```python
df = df[(df["Trddt"] - df["ListDt"]).dt.days >= 365]
```

## 规则 5：B 股 / H 股双重上市

A 股股票代码规则（沪深主板）：
- `60xxxx` — 上海主板 A 股
- `00xxxx` — 深圳主板 A 股
- `30xxxx` — 创业板
- `68xxxx` — 科创板
- `90xxxx` — 上海 B 股（人民币标价、外币交易）
- `20xxxx` — 深圳 B 股

**默认做法**：只保留 A 股普通股（首位为 0/3/6 但排除 9）。

```stata
keep if substr(Stkcd, 1, 1) == "0" | substr(Stkcd, 1, 1) == "3" | substr(Stkcd, 1, 1) == "6"
* 进一步排除特殊代码段
drop if substr(Stkcd, 1, 2) == "20"  // 深圳 B 股
drop if substr(Stkcd, 1, 2) == "90"  // 上海 B 股
```

**A+H 双重上市**（如工商银行 601398 + 1398.HK）：研究设计若不区分 A/H 股流动性，可保留 A 股部分；若研究跨市场套利则需引入港交所数据。

## 规则 6：关键变量缺失值处理

**原则**：实证经管论文 **不插补关键变量**（如净利润、ROA、营业收入）。

| 变量类型 | 处理方式 |
|---|---|
| 因变量（被解释变量） | 缺失则删除 |
| 关键自变量 | 缺失则删除 |
| 控制变量（公司规模、杠杆率等） | 缺失则删除 |
| 治理变量（独董占比等） | 缺失可考虑用前一年值、若缺失年份过多则删除 |
| 行业代码、地区代码 | 不应缺失；缺失则删除 |

**Stata**：
```stata
* 删除关键变量任一缺失
drop if missing(roa, lev, size)
```

**Python**：
```python
df = df.dropna(subset=["roa", "lev", "size"])
```

> **永远不要**用 0 填补盈利类指标缺失值——会人为制造大量"零盈利"虚假信号。

---

## 规则 7：极值缩尾（详见 `winsorize-standard.md`）

主流做法：**全样本对所有连续变量按 1% / 99% 分位数缩尾**。

```stata
ssc install winsor2
winsor2 roa lev size growth, replace cuts(1 99)
```

---

## 规则 8：样本期窗口

| 起点 | 论据 |
|---|---|
| 1990 | 沪深交易所成立。但早期数据稀疏 + 制度不规范 |
| 1998 | 多数论文起点（A 股已成型，制度初步稳定） |
| 2003 | 公司治理报告 (`CG_Ybasic`) 系统披露起点 |
| 2007 | 新会计准则实施起点（IFRS 趋同），财务数据可比性最高 |

**实操建议**：研究主题涉及财务报表细节 → 起点不早于 2007；只用市场行情 → 可上溯到 1998。

## 完整最小样本筛选模板（Stata 伪代码）

```stata
use raw_panel.dta, clear

* 1. 时间窗口
keep if year >= 2007 & year <= 2023

* 2. 行业筛选
drop if substr(IndCd, 1, 1) == "J"      // 剔金融
drop if substr(IndCd, 1, 1) == "K"      // 剔房地产（可选）

* 3. ST / PT
drop if Trdsta != 1

* 4. IPO 当年
drop if (year - year(ListDt)) < 1

* 5. B 股
drop if substr(Stkcd, 1, 2) == "20" | substr(Stkcd, 1, 2) == "90"

* 6. 关键变量缺失
drop if missing(roa, lev, size, growth)

* 7. 缩尾
winsor2 roa lev size growth bm, replace cuts(1 99)

save clean_panel.dta, replace
```

执行完上述 7 步后，得到的样本就是 A 股实证经管论文的"标准默认样本"。
