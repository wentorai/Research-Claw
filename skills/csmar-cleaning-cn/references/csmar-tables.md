# CSMAR 高频用表清单

本文件列出实证经管研究中**最高频被使用**的 CSMAR 库表。字段命名以 CSMAR 数据手册 (CSMAR Data Manual) 公开版本为准，**遇不一致以最新版数据手册为准**。

---

## 一、股票交易类（市场行情数据库）

### 1.1 `TRD_Dalyr` — 个股交易日数据

A 股最常用的日度行情表，事件研究 / 股价同步性 / 流动性研究的基本输入。

| 字段 | 含义 |
|---|---|
| `Stkcd` | 股票代码（6 位字符串，主键之一） |
| `Trddt` | 交易日期（YYYY-MM-DD） |
| `Opnprc` | 开盘价 |
| `Hiprc` | 最高价 |
| `Loprc` | 最低价 |
| `Clsprc` | 收盘价 |
| `Dnshrtrd` | 日个股交易股数 |
| `Dnvaltrd` | 日个股交易金额 |
| `Dsmvosd` | 流通市值（千元） |
| `Dsmvtll` | 总市值（千元） |
| `Dretwd` | 考虑现金红利再投资的日个股回报率 |
| `Dretnd` | 不考虑现金红利的日个股回报率 |
| `Trdsta` | 交易状态（1=正常上市，2=ST，3=*ST，4=暂停上市，etc.；具体编码以最新手册为准） |
| `Markettype` | 市场类型（1=上交所主板，4=深交所主板，16=创业板，64=科创板，等） |

### 1.2 `TRD_Mnth` — 个股交易月数据

月度版，资产定价 / 因子构造常用。

| 字段 | 含义 |
|---|---|
| `Stkcd` | 股票代码 |
| `Trdmnt` | 交易月份（YYYY-MM） |
| `Mclsprc` | 月收盘价 |
| `Mretwd` | 考虑现金红利再投资的月个股回报率 |
| `Mretnd` | 不考虑现金红利的月个股回报率 |
| `Msmvosd` | 月末流通市值（千元） |
| `Msmvttl` | 月末总市值（千元） |
| `Mnshrtrd` | 月个股交易股数 |

### 1.3 `STK_MKT_DalyR` — A 股市场日度回报

含市场组合（流通市值加权 / 等权）日收益，用于 CAR / 三因子模型基准。

| 字段 | 含义 |
|---|---|
| `Trddt` | 交易日期 |
| `Markettype` | 市场分类 |
| `Cdretwdos` | 考虑现金红利再投资的流通市值加权平均日市场回报率 |
| `Cdretmdos` | 考虑现金红利再投资的总市值加权平均日市场回报率 |
| `Cdreteqos` | 考虑现金红利再投资的等权平均日市场回报率 |

## 二、财务报告类（公司财务数据库）

### 2.1 `FS_Combas` — 资产负债表（合并报表）

最常用的财务表之一。字段以 `A` 开头 + 9 位编码。

| 字段 | 含义 |
|---|---|
| `Stkcd` | 股票代码 |
| `Accper` | 会计期间结束日（YYYY-MM-DD，如 2020-12-31 表示年报） |
| `Typrep` | 报表类型（A=合并报表，B=母公司报表） |
| `A001000000` | 资产总计 |
| `A001100000` | 流动资产合计 |
| `A001101000` | 货币资金 |
| `A002000000` | 负债合计 |
| `A002100000` | 流动负债合计 |
| `A003000000` | 所有者权益合计 |

> 资本结构变量构造：`Lev = A002000000 / A001000000`。

### 2.2 `FS_Comins` — 利润表（合并报表）

字段以 `B` 开头。

| 字段 | 含义 |
|---|---|
| `Stkcd` | 股票代码 |
| `Accper` | 会计期间结束日 |
| `Typrep` | 报表类型 |
| `B001100000` | 营业收入 |
| `B001200000` | 营业成本 |
| `B001300000` | 营业税金及附加 |
| `B001400000` | 销售费用 |
| `B001500000` | 管理费用 |
| `B001600000` | 财务费用 |
| `B001700000` | 研发费用（部分年度） |
| `B001000000` | 营业利润 |
| `B002000000` | 净利润 |

### 2.3 `FS_Comscfd` — 现金流量表（合并直接法）

字段以 `C` 开头。

| 字段 | 含义 |
|---|---|
| `Stkcd` | 股票代码 |
| `Accper` | 会计期间结束日 |
| `C001000000` | 经营活动产生的现金流量净额 |
| `C002000000` | 投资活动产生的现金流量净额 |
| `C003000000` | 筹资活动产生的现金流量净额 |

### 2.4 `FN_Fn041` — 财务指标—相对价值类

预算好的常用比率，省去手工构造步骤。

| 字段 | 含义 |
|---|---|
| `Stkcd` | 股票代码 |
| `Accper` | 会计期间结束日 |
| `F040401A` | 总资产报酬率 ROA |
| `F040402A` | 净资产收益率 ROE |
| `F040701A` | 每股收益 EPS |
| `F040601A` | 营业利润率 |
| `F040801A` | 总资产周转率 |

> CSMAR 财务指标分多张表 (`FN_Fn001` 到 `FN_Fn059` 等)；具体口径以数据手册为准。

## 三、公司基本信息

### 3.1 `EN_EnterpriseInfo` — 上市公司基本信息

每只股票一条记录的截面表。

| 字段 | 含义 |
|---|---|
| `Stkcd` | 股票代码 |
| `Stknme` | 股票简称（最新） |
| `IndCd` | 行业代码（CSRC 2012 大类，单字母 A–S） |
| `IndNme` | 行业名称 |
| `ListDt` | 上市日期 |
| `EstDt` | 公司成立日期 |
| `Provcd` | 注册地省份代码 |
| `Citycd` | 注册地城市代码 |
| `Ofcadd` | 办公地址 |
| `Repaddr` | 注册地址 |

### 3.2 `TRD_Co` — 上市公司股票代码及简称变更表

追踪 ID 变更的关键表（详见 `id-tracking.md`）。

| 字段 | 含义 |
|---|---|
| `Stkcd` | 股票代码 |
| `Stknme` | 简称 |
| `Listdt` | 该简称启用日期 |
| `Delistdt` | 该简称结束日期 |

## 字段名稳定性提醒

CSMAR 偶尔会调整字段命名（特别是新增子项时编号会变）。**以最新版数据手册为准**。论文中引用字段时，建议显式注明使用的数据版本（如 "CSMAR v2024Q3"）以保证可重复性。
