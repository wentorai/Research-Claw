# Prompt: 重要性 / 立项依据 / Significance 撰写

## 用途

帮助用户撰写各类项目"重要性"段落。统一回答 **"so what"**，但按目标项目体例呈现。这是 reviewer 一份本子最先读的部分（NIH 是 Specific Aims 第 1 段；NSF 是 Project Summary Overview；ERC 是 Synopsis 第 1 段；国家社科是选题依据第 1 段）。**前 30 秒赢不了 reviewer，整本就难翻盘**。

## 项目体例映射

| 项目 | Section 名 | 篇幅指引 |
|------|-----------|---------|
| 国家社科基金 | "选题依据"中的"研究意义" | 占活页约 1/4，~1500-2500 字 |
| 教育部人文社科 | "选题意义" | ~1000-2000 字 |
| NIH R01 | Specific Aims Para 1 + Significance（Research Strategy 内 2-3p） | 1 页 + 2-3 pages |
| NSF | Project Summary Overview + Project Description Background | 1/3 页 + 1-2 pages |
| ERC | Extended Synopsis 第 1 段 + B2 状态艺术 | 0.5-1 page synopsis 内 |
| MSCA PF | Excellence section 第 1 段（"State of the Art and Objectives"） | ~1 page |

## 输入要求

询问用户：

1. **目标项目类型**（决定体例）
2. **研究问题**（≤ 50 字 — 一句话回答得出）
3. **本研究服务的真问题**（疾病负担 / 政策痛点 / 学科 gap / 国家战略）
4. **本研究的拟突破点**（与现有文献相比能 contribute 什么）
5. **拟引用的关键文献**（≥ 5 条 DOI / 引文条目，否则提醒用户先做文献调研）
6. **是否有 preliminary data / pilot study**（决定能否用"leverages our preliminary data"这类高分短语）

## 通用结构（统一回答 5 个问题）

无论目标项目体例，"重要性"段落本质上回答 5 个问题（"5 W"）：

1. **Why** is this topic important?（领域宏观 / 社会现实意义）
2. **What** is currently known?（已有共识 + 已有方法）
3. **What's missing**?（critical gap = 卡住领域前进的 specific barrier）
4. **How** does this project address the gap?（具体到本项目的 contribution）
5. **What's the impact** if successful?（学术 / 应用 / 政策 / 临床）

写作时按目标体例**重新分配**这 5 个问题的篇幅。

## "30 秒 hook" 句式（最关键的开篇句）

reviewer 只用 30 秒看你的开篇——必须用一句话把"问题严重性 + gap"钉住。

### NIH 模式（疾病 / 临床导向）

```
Despite [decades of advance / billions in investment / current standard of care], 
[clinical outcome] remains [problem statement with number]. 
The mechanism by which [X causes Y] remains unknown, 
constituting a critical barrier to [therapeutic / preventive] progress.
```

**例**：
> *Despite the introduction of immune checkpoint inhibitors, fewer than 30% of patients with melanoma achieve durable response. The mechanism by which tumor-intrinsic [X] confers resistance remains poorly defined, constituting a critical barrier to next-generation immunotherapy.*

### NSF 模式（基础科学 / 工程导向）

```
[Field] has long assumed that [prevailing assumption]. 
However, recent [observation / advance] suggests that [counter-evidence], 
opening a fundamental question: [reframed question]. 
Answering this question is essential to [advancing knowledge / informing practice].
```

**例**：
> *Algorithmic decision-making research has long assumed that fairness and accuracy involve a fundamental tradeoff. However, recent theoretical advances suggest this tradeoff may be a consequence of model class restrictions rather than a deep truth. This raises a fundamental question: under what conditions does the tradeoff vanish?*

### ERC 模式（ground-breaking / frontier 导向）

```
Current [field] research treats [phenomenon X] and [phenomenon Y] as separate. 
This separation is a historical artifact, not a scientific necessity, 
and it prevents us from addressing [question Z] —— a question whose answer would 
[transform the field / open a new line of inquiry].
```

**例**：
> *Current cognitive neuroscience treats perception and action as functionally distinct, separated by central decision-making. Recent evidence challenges this separation but no unified framework yet exists. Constructing such a framework is the precondition for understanding flexible behavior in dynamic environments.*

### 中文资助模式（国家社科 / 教育部 / 省部级）

```
[宏观背景一句]。[领域共识一句]。然而，[关键不足或矛盾一句]，
[核心研究问题以问句方式直接抛出]？这一问题不仅关乎 [理论推进]，
更是 [国家战略 / 实践需求] 中亟待解答的瓶颈。
```

**例**：
> *新质生产力的提出标志着中国高质量发展进入新阶段。学界普遍认为新质生产力以技术革命性突破为核心特征。然而，新质生产力的形成机制、测度方法及对区域经济差距的影响尚未系统厘清——新质生产力究竟通过何种机制重构区域经济格局？这一问题不仅关乎"新发展阶段"的理论建构，更是"区域协调发展""共同富裕"重大战略中亟待解答的关键。*

## 体例特化输出模板

### 国家社科 / 教育部人文社科

```
[第 1 段] 选题的理论价值
- 学术史脉络（中外对话）
- 本研究在学科理论上的推进点
- 与具体理论框架 / 学派对话（具名 ≥ 2 个流派或代表学者）

[第 2 段] 选题的应用价值
- 服务国家战略 / 政策需求 / 社会发展
- 转化路径（学术 → 政策 → 实践）
- 量化承诺（"研究报告报送 [具体决策部门]"）

[第 3 段] 国内外研究现状述评
- 国外动态（≥ 50% 国际文献）—— 用"综 / 梳 / 述 / 评"四要素
- 国内进展
- 研究空缺（critical gap，**这一段是文献综述的点睛之笔**）
```

### NIH Significance

```
[结构化 sub-section headers]
1. Magnitude of [疾病 / 现象]                                  (~0.5 page)
   - 流行病学 + 经济负担数据 + 1 张 trend figure
2. Current State of the Field                                  (~0.5 page)
   - what is known + what is NOT known
3. Critical Barriers                                           (~0.5 page)
   - 2-3 个 specific barriers（每个粗体一句话）
4. How This Project Addresses the Barriers                     (~0.5 page)
   - aim 与 barrier 一一映射
5. Significance of Expected Outcomes                           (~0.3 page)
6. Premature Truncation Avoidance                              (~0.2 page)
   - 一句话：本项目是 PI 长期 program 的合理一步
```

### NSF Intellectual Merit (Project Description 内)

```
- 用 Project Description 的 Background / Motivation 段开篇
- 与 Significance 类似，但要明确链接到 "advance knowledge"
- 不要混入 broader impacts 的内容
- 显式标注 "Intellectual Merit" subsection header
```

### ERC Extended Synopsis 第 1 段

```
- 高度凝练：1-2 段说清"state of the art + critical question + ambition"
- 强调 ground-breaking 不是 incremental
- 显式回应"why this is high-risk-high-gain not a safe bet"
- 引用 ≤ 15 篇关键文献
```

## "Reviewer 一定会问的 5 个问题"自检

写完 significance 段，**自己扮演 reviewer**逐条问：

1. **"So what?"** —— 我能用 1 句话答出"为什么领域必须解决这个 gap"吗？
2. **"Why now?"** —— 为什么是现在？什么 enabling technology / data / theory 让这个 question 现在可解？
3. **"Why you?"** —— 为什么是这位 PI？preliminary data / unique cohort / unique method 在哪里？
4. **"Why this approach?"** —— 为什么 specific aims 是回答 gap 的最佳路径，而非其他？
5. **"Why should I trust this won't be just another paper?"** —— 项目结束后领域会改变什么？什么 indicator 能验证 transformative impact？

每问都要在 significance 内**至少有 1 句对应回答**。

## Worked Example 1: NIH R01 — 五段式 Specific Aims

**研究问题**：靶向 PD-1 通路在三阴性乳腺癌（TNBC）中疗效有限，原因不明。

```
SPECIFIC AIMS

[Para 1 — Hook + Gap]
Despite immunotherapy revolutionizing oncology, fewer than 25% of patients 
with metastatic triple-negative breast cancer (TNBC) achieve durable response 
to PD-1 blockade [Mittendorf 2024]. Tumor-intrinsic features [X, Y, Z] have 
been implicated in primary resistance, but the role of [novel mechanism A] 
remains uncharacterized. *This represents a critical barrier to next-generation 
immunotherapy in TNBC.* The long-term goal of our laboratory is to define 
tumor-intrinsic determinants of immunotherapy response in breast cancer.

[Para 2 — Hypothesis + Rationale]
The objective of this proposal is to determine how [novel mechanism A] regulates 
PD-1 blockade response in TNBC. **Our central hypothesis is that [A] regulates 
[CD8+ T cell infiltration] via [pathway B], and that pharmacologic inhibition of 
[A] sensitizes resistant tumors to PD-1 blockade.** This hypothesis is supported 
by our preliminary data showing (a) elevated [A] expression in non-responders 
(Fig 1A); (b) [A] knockdown restores T-cell infiltration in PDX models (Fig 1B).

[Para 3 — Aims]
Aim 1. Determine the molecular mechanism by which [A] suppresses [B].
   Working hypothesis: [A] directly binds [protein] to inhibit [pathway].
   Approach: ChIP-seq + biochemical reconstitution in TNBC cell lines.
   Expected outcome: identification of [A]-[protein] regulatory module.

Aim 2. Test whether [A] inhibition restores anti-tumor immunity in vivo.
   Working hypothesis: pharmacologic [A] inhibitors enhance PD-1 efficacy 
   in PDX-derived TNBC models.
   Approach: 4 PDX models × 4 arms (vehicle / αPD-1 / [A]-i / combo); RNA-seq + flow.
   Expected outcome: causal evidence that [A] inhibition is required.

Aim 3. Identify clinical predictors of [A]-driven resistance.
   Working hypothesis: a [A]-pathway signature predicts non-response in 
   patients receiving αPD-1 monotherapy.
   Approach: spatial transcriptomics on N=120 retrospective tumor samples.
   Expected outcome: a clinically actionable biomarker signature.

[Para 4 — Innovation]
This proposal is innovative both conceptually (first to test [A] as 
immunotherapy regulator in TNBC) and methodologically (combining PDX 
pharmacology with patient spatial transcriptomics).

[Para 5 — Impact]
Successful completion will (a) define a new resistance axis in TNBC; 
(b) provide preclinical rationale for combination trials; (c) yield a 
biomarker for patient stratification — directly informing clinical 
trial design within 5 years.
```

## Worked Example 2: 国家社科基金一般项目 — 立项依据

**研究问题**：数字平台经济中的算法歧视与劳动者权益保护。

```
一、选题依据

(一) 选题的理论价值

党的二十大报告强调"加强数字经济治理"，"维护新就业形态劳动者合法权益"。
然而，平台算法对劳动者的派单、定价、考核构成了一种新型的"技术-劳动"关系，
现有法学劳动关系理论与社会学组织理论难以充分解释。本研究的理论价值在于：
**第一**，在新就业形态语境下重构"算法权力"概念，与法兰克福学派的"工具理性"
批判和 Lawrence Lessig 的"代码即法律"命题对话；**第二**，引入 Coase 交易
费用框架重新阐释平台劳动关系的边界，回应 Acquier 等 (2017) "组织化平台"
理论的不足；**第三**，构建"算法歧视识别 - 维权机制 - 治理路径"三层分析框架。

(二) 选题的应用价值

本研究直接服务于：(1) 中央"加强数字经济治理"的战略部署；(2)《新就业形态
劳动者权益保护办法》的实施落地；(3) 全国总工会"新就业形态劳动者建会入会"
工作。预期形成**研究报告 1 份报送人社部劳动关系司**，**专著 1 部 (≥ 30 万字)
**，**CSSCI 论文 5 篇**。

(三) 国内外研究现状述评

【综】算法与劳动关系研究自 2018 年起进入爆发期。Web of Science 数据库
显示 2019-2024 年相关英文文献年均增长 24%，CSSCI 数据库年均增长 31%，
研究主题从"算法管理"逐步延伸至"算法歧视""算法问责"。

【梳】国际研究可分三个流派：
(1) 平台治理流派 (Schor 2020; Cant 2020)：聚焦平台企业的算法权力运作；
(2) 算法公平流派 (Barocas & Selbst 2019)：从计算机科学视角识别算法歧视；
(3) 劳动法学流派 (Stewart & Stanford 2017)：聚焦劳动者法律地位重塑。

国内研究可分三阶段：…（略）

【述】国际三个流派各有所长但各自为政：平台治理流派擅长描述但缺乏
量化识别；算法公平流派擅长识别但脱离劳动情境；劳动法学流派擅长
规范分析但难以触及算法黑箱。

【评】**关键不足**：(1) 缺乏跨流派的整合框架；(2) 国内研究以理论引介
为主，基于中国本土平台数据的实证研究极少；(3) 维权机制层面的实证
评估几乎空白。本研究将填补这三类空缺。
```

## 反例（每种体例都常见的低分写法）

- **空泛宏大**：「本研究对推动 XX 学科发展具有重要意义」（没具体到推动什么、与谁对话）
- **罗列文献**：连续 10 个 "[Smith 2020] 发现 ... [Lee 2021] 发现 ..."，没有评述
- **悲剧叙事**：「X 病每年死 X 人 → 所以我做这个」，缺乏机制层面的 gap
- **自我抬高**：「本研究是首次 / 国际领先」 — 必须有 specific comparator 才能说「首次」
- **跨学科借词**：把数学 / 物理 / 计算机的术语生硬塞入社科段落
- **"重大意义"句**：中文常见"具有重大理论价值与现实意义"——这是 placeholder 不是论证

## 工作流

1. 询问目标项目 + 6 个输入信息
2. 检查文献数量 / 质量；如不足主动建议调用 `research-deep` Skill 先补
3. 按目标体例产出**结构化骨架**（带 sub-section header）+ **每段第 1 句**
4. 把"30 秒 hook"放在最显眼位置（NIH Aims Para 1 / NSF Overview / ERC Synopsis 首段 / 中文立项依据首段）
5. 用户填充后帮助 polish（保留原意，提升论证密度）
6. 最后做"reviewer 5 问"自检
