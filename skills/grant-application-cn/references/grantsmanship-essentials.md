# Grantsmanship Essentials — 跨基金的通用写作原则

不论申请 NIH R01、NSF CAREER、ERC Starting、国家社科还是 MSCA，资助申请书的"成功率天花板"都受同一组写作原则约束。各基金的体例差异是表面，**底层"grantsmanship"是共通的**。本文整合了公开的 NIH OER training、NSF reviewer 指南、Robert Porter / Roger Lipke / writedit / Bouvier Grant Group 等公开资料中的共识，归纳为 **8 条核心原则**。

> 本文与 `references/nih-r01.md`、`nsf-broader-impacts.md`、`erc-cv-and-track-record.md`、`nssf-guide.md` 互补——这 4 份文件讲"各基金的体例与雷区"，本文讲"无论哪个基金都通行的写作哲学"。

## 原则 1: Hook —— 30 秒赢得 reviewer 才有第二口气

**所有 reviewer 都在飞机上 / 周末 / 临提交前一晚读你的本子**。前 30 秒（NIH 的 Specific Aims 第 1 段；NSF 的 Project Summary Overview；ERC 的 Synopsis 首段；国家社科的立项依据首段）必须钩住——否则 reviewer 已经形成"这是 borderline"的印象，后面再精彩也难翻盘。

### Hook 的 3 个层次

```
1. Surprise / Tension（震撼或矛盾）
   - "Despite [decades of advance], [outcome] remains [problem]."  
   - "Field has long assumed X. Recent evidence challenges X."

2. Specificity（具体到一个可视化的事实）
   - 一个数字："fewer than 25% of patients..."
   - 一个机制："the binding affinity of A to B is..."
   - 一个场景："when a clinician makes a diagnosis without [...]"

3. Stakes（如果不解决会怎样）
   - 临床后果 / 政策瘫痪 / 学科停滞 / 国家安全
```

### 反例

- "Cancer is a leading cause of death." —— 陈词滥调，0 信息量。
- "数字经济发展迅速。" —— 国家级套话，0 论证。
- "AI is transforming science." —— ESG / 营销语言，不是 grant hook。

### 写作技巧

写完整本后回头**重写 hook 至少 3 次**，每次时间间隔 ≥ 1 天。最终版本应满足"PI 在电梯里被 program officer 问到时能脱口而出"。

## 原则 2: Specific Is Better —— 每一个 claim 都用 specific 替代 generic

reviewer 给低分的最常见原因之一就是 "vague" / "lacks specificity"。

| Generic（低分） | Specific（高分） |
|----------------|---------------|
| "We will study X" | "We will determine whether X regulates Y by binding Z" |
| "Multiple methods" | "Single-cell RNA-seq + STED super-resolution + CRISPRi screen" |
| "Large sample size" | "n = 1,200 (power 0.85 for d = 0.3, α = 0.05)" |
| "Novel approach" | "first to integrate [A] with [B] in [scope]" |
| "Will impact education" | "deploys 2-week curriculum module to 600 K-12 students/yr at 4 Title-I schools" |
| "Improve outcomes" | "reduce time-to-diagnosis by 40% (baseline 14 days → target 8.4 days)" |
| "重要意义" | "可为 [具体决策部门] 制定 [具体政策] 提供 [具体支持]" |

### 自检：每读一句问"how would I measure this?"

如果 claim 不能映射到一个**可量化、可观察、可验证**的指标，重写。

## 原则 3: Show Preliminary —— Preliminary Data 双重作用

Preliminary data 在大多数高分本中扮演**两个不同角色**：

### 角色 A: "The hook" —— 一项震撼的初步发现

一张图能让 reviewer "sit up"：unexpected observation / dramatic effect / beautiful proof-of-concept。这张图通常出现在 Significance 末尾或 Approach 开头。

### 角色 B: "The feasibility evidence" —— 技术可行性的实锤

干净的 western blots / 验证过的工具 / 已搭建的 pipeline / 已签订的 cohort access agreement——证明你**能做**你说要做的事。

### 关键认识：Preliminary data 的目的不是赢辩论，是阻止辩论开始

> *"The goal isn't to win a debate—it's to prevent one from starting."*

每个 reviewer 心里都有一个"如果做不出来怎么办"的疑虑。强 preliminary data 让这个疑虑根本起不来。**没有 prelim 的 R01 / 没有 pilot 的国家社科重点**，无论多漂亮的论证都难逃 "speculative"批评。

### 如何展示 preliminary data

每段 prelim 必须含 4 件套：

1. **Figure 引用**（"Fig 1A shows..."）
2. **Methods 简述**（n = X, statistical test used）
3. **Conclusion**（**用粗体一句话**："**These data suggest that X regulates Y.**"）
4. **与 specific aim 的连接**（"This finding directly motivates Aim 2."）

> R21 不强制 prelim，但有 prelim 的 R21 仍比纯思辨高分。

## 原则 4: Show Track —— 用 PI 的过往证明能 deliver 未来

无论体例如何，**PI 的 track record 是 reviewer 评估"这本子能否兑现"的核心信号**。

### 4 个 framing 技巧

1. **聚焦 contributions, not topics**：不是说 "I work on X"，而是 "I established Y in X" / "I challenged the prevailing view that Z"
2. **Independent assessment**：让第三方说话——"cited 800 times" / "featured in [F1000 / Nature highlight]" / "adopted by [N independent groups]"
3. **Trajectory（轨迹）而非 inventory（清单）**：让 reviewer 看到**方向感**——10 年前关注 A → 5 年前转向 B → 现在准备做 C —— 项目是 trajectory 的合理下一步
4. **Distinguish independent vs collaborative work**：在 ERC StG/CoG 尤其关键

### 反例

- "I have published 80 papers" —— 数量没意义，故事才有意义
- "My h-index is 32" —— 数字本身在 ERC / NIH 评审中不是论据
- "We are leaders in this field" —— 自封的 leader 在 reviewer 眼中减分

## 原则 5: Dispel Reviewer Concerns —— 主动回应"如果失败怎么办"

reviewer 写 critique 时最常用的几句模板：

- *"The authors have apparently not considered the possibility of failure."*
- *"What happens if Aim 1 does not yield the expected results?"*
- *"The proposal lacks contingency planning."*
- *"Statistical methods are inadequate for the proposed sample size."*

### Pre-empt 这些 critique 的写作技巧

每个 aim 末尾必须有 **"Potential Pitfalls and Alternative Strategies"** 段（即使体例不强制）：

```
Pitfall 1: [明确预想的失败模式]
Likelihood: low / medium / high
Alternative: [备选方案，最好引一篇 PI 自己的 prior work / 已有 collaborator]

Pitfall 2: ...
```

### 用 "Although..., we will..." 句式系统消解疑虑

> *"Although [reviewer concern], we will [mitigation], as evidenced by [supporting data / track record]."*

这类句子直接 disarm 心理疑虑。

### 在 Significance / Approach 中**主动引用**与你立场不同的论文

- 如果某篇论文的发现挑战你的 hypothesis，主动 cite + 解释为什么本研究 reframe 了那个矛盾
- **不引用 = 装作不知道**，reviewer 知道这篇论文在；他会因为你"装作不知道"扣分

## 原则 6: Reviewer-friendly Layout —— 视觉上让 reviewer 5 秒定位关键句

reviewer 一份本子读 30 分钟。**视觉结构**直接影响 review 质量。

### 排版必备

1. **Sub-section headers**：**粗体** + 数字编号（"1. Significance"，"Aim 1. Determine..."）
2. **Topic sentence first**：每段第一句是主旨句，即使 reviewer 只读首句也能 follow
3. **关键概念加粗**——但每段最多 1 处，不要全粗（视觉无锚）
4. **Bulleted list** 替代连续散文（特别在 Approach / pitfalls / rigor 段）
5. **Figure with self-contained caption**：每张图的 caption 应能独立读懂（一句话结论 + n + statistical test）
6. **White space**：每 4-5 行就分段；不要"段落墙"

### 反例

- 一整页的 dense 散文 —— reviewer 心理疲劳
- 关键句没有视觉锚 —— reviewer 找不到 "the punchline"
- Figure caption 仅写"Distribution of X" —— reviewer 读不懂图

## 原则 7: Trust the Reviewer's Mental Model —— 用他熟悉的 frame 包装你的工作

每个 reviewer 都有他自己的 mental model（领域经典论文 / 经典框架 / 常见 pitfall）。**把你的工作 frame 进他熟悉的 mental model**，再展示创新。

### 例：跨学科本子的"双语"frame

如果你的本子从 social network analysis 出发研究 epidemiology：
- **对 epidemiology reviewer**: frame 为 "extension of compartmental models with explicit network structure"
- **对 SNA reviewer**: frame 为 "validation of network topology metrics using disease spread as ground truth"

跨学科本子的常见踩雷：在两个领域分别写"introduction"但不互译——结果两个 reviewer 都觉得"作者不懂我的领域"。

### 例：选 study section / panel 时主动迎合

NIH 允许 PI 在 cover letter 推荐 study section。**选与你 mental model 最相近的 panel**——同一份本子在 IMM (Immunology) panel 和在 CSR (Cell biology) panel 的命运可能截然不同。

## 原则 8: Iterate Ruthlessly —— Grant 是一个 iteration 工艺品

强 PI 的本子都经过：
- **≥ 3 次 internal review**（自己 / mentor / 1-2 位 senior collaborator）
- **≥ 1 次 mock study section**（找 3-4 位非本领域的同事扮演 reviewer 模拟评审）
- **6 个月以上的 prep time**（不是 deadline 前 2 周冲刺）

### 写作 iteration 的最少 3 步

1. **First draft**：把所有想法写下来，不管结构和长度
2. **Structure pass**：按目标体例（Aims / Significance / Approach）重组；删掉 50%
3. **Polish pass**：每段重写"topic sentence"；用"Although..., we..." 句式 disarm critiques

### "Reviewer 5 问"自检

每写完一个 section，扮演 reviewer 问 5 个问题：

1. **So what?** —— 解决这个 gap 让领域怎样？
2. **Why now?** —— 为什么现在能解？
3. **Why you?** —— 为什么是你？
4. **Why this approach?** —— 为什么不是其他方法？
5. **Why should I trust this won't be incremental?** —— 项目结束后领域改变什么？

每问应在本子内有 ≥ 1 句对应回答。

## 跨基金的"高分短语库"

下表整合 NIH / NSF / ERC / 国家社科评审中**反复出现于被资助本**的"信号词"。**不是堆砌**，每段顶多 1-2 个。

| 短语 | 适用基金 | 暗示信号 |
|------|---------|---------|
| "rigor and reproducibility" | NIH (强制) / NSF | 合规 + 严谨 |
| "premature truncation" | NIH | 暗示资助的必要性 |
| "leverages our preliminary data" | NIH / NSF / ERC | 强可行性 |
| "leverages a unique [cohort/dataset]" | NIH / NSF / ERC | 独占性 |
| "ground-breaking nature" | ERC (强制) | 频率词 |
| "high-risk-high-gain" | ERC | 与 incremental 区隔 |
| "trans-disciplinary" / "convergent" | NIH Common Fund / NSF | 跨学科加分 |
| "addresses a critical barrier" | NIH | 显式回应 NIH "barrier" 框架 |
| "sex as a biological variable" | NIH (强制 from 2016) | 合规 |
| "specific, desired societal outcomes" | NSF (PAPPG 24-1+) | 显式 BI 框架 |
| "integrated research and education" | NSF CAREER (强制) | CAREER 关键词 |
| "informed by stakeholder input" | NSF / NIH (health equity) | community-engaged 加分 |
| "中国自主知识体系" | 国家社科 (近年高频) | 政治正确性 + 学科建设 |
| "服务国家战略" | 国家社科 / 教育部 | 应用价值 |
| "学术史脉络" | 国家社科 | 文献综述深度 |

## 综合自检 Checklist —— 提交前的最后一遍

跨基金通用：

- [ ] **Hook**：前 30 秒能让 reviewer 想读下去？
- [ ] **Specific**：每个 claim 都能映射到可量化指标？
- [ ] **Preliminary**：有"hook prelim"+"feasibility prelim"双重？
- [ ] **Track**：PI 的 trajectory 是否能让 reviewer 看到"必然走到本提案"？
- [ ] **Pitfalls**：每个 aim / WP 含主动 pitfalls + alternatives？
- [ ] **Layout**：sub-headers + bold + bullets + figures with self-contained captions？
- [ ] **Frame**：用 reviewer 熟悉的 mental model 包装跨学科创新？
- [ ] **Iteration**：≥ 3 次 internal review + ≥ 1 次 mock？
- [ ] **5 问**：so what / why now / why you / why this / why transformative 各有答案？
- [ ] **高分短语库**：本基金对应的"信号词"已至少出现 3 个（不堆砌）？
