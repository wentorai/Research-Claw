# 国际资助评审心理学：NIH / NSF / ERC / MSCA / 国内匿名评审

不同资助机构的"评审心理"差异极大。同一份本子按 NIH 体例写完直接投 ERC，几乎必拒；NSF 的 reviewer 心理与 NIH 的 study section 完全不同。本文件梳理**评审专家在制度之外的真实行为模式**——这是把"形式合规"的本子变成"高分入选"本子的关键。

> 本文件源于 NIH CSR 公开手册、NSF Merit Review 公开指南、ERC 评审指南、MSCA Guide for Applicants、各机构公开的 Funding Opportunity Statistics、以及多位获资助 PI / 评审专家在公开博客 / 学协会培训中的分享整合而成。**最终请以当年 RFA / Call 公布的最新文件为准**。

---

## 一、NIH Study Section Dynamics（R01 / R21 / R03 等）

### 1.1 Triage 机制：~50% 申请被 Not Discussed (ND)

NIH study section 一般 25-35 名 reviewer，每轮要审 60-100 份本子。**制度上**每份分给 3 位 primary/secondary/tertiary reviewer 打分（1-9 分制，1 是最好），**实际上**：

- 全部本子先**预打分**（preliminary impact score）；
- **打分排在最后 50%** 的本子被 **Not Discussed**——直接终结，不进入会议讨论；
- 进入讨论的 ~50% 本子在会上重新打分，最终给出 **impact score (1-9, 整数 ×10 = 10-90)** 和 **priority percentile**（在所有讨论本子中的百分位排名）。

**写作启示**：
- 你的本子要先**通过 triage 这一关**——意味着 3 位 primary reviewer 中**任何一位**在前 30 分钟没看到亮点，本子可能直接 ND；
- ND 的本子**收不到详细评语**——只有 summary statement 顶部一句 "not discussed by the panel"，对 PI 改投帮助极小。

### 1.2 9 分制评分实际分布

| Impact Score | Priority Percentile | 含义 |
| --- | --- | --- |
| 1.0-1.5 | 1-3% | 极少（每个 study section 一轮可能 0-1 个） |
| 1.5-2.0 | 3-10% | **Fundable**（高于 payline 安全区） |
| 2.0-3.0 | 10-25% | **Fundable border**（payline ~10-15% 时进入灰区） |
| 3.0-4.0 | 25-50% | 讨论但**远低于 payline** |
| 4.0+ | 50%+ | 几乎不会被资助（除非 program officer 强烈干预） |

NIH 各 institute 的 **payline 不同**：NCI / NHLBI 通常 ~10-12%；NIA / NIDDK 在某些方向到 ~15%；R21 通常无 percentile，按 impact score < 3 内部决策。

**写作启示**：
- 投本子前查目标 institute 当年的 payline（NIH RePORTER 公开），**心理预期定在 percentile 10% 以内**；
- 2.5 分本子 ≈ "高度有潜力，但需要重大修改"——往往需要 A1 (resubmission) 才能拿到资助。

### 1.3 Reviewer 30 秒注意力机制

NIH reviewer 平均每份本子分配 ~2-4 小时（深读阶段），但**第一轮浏览**只有 **15-30 分钟**——决定预打分：

1. **Page 1 (Specific Aims)**：前 60 秒决定整体观感；
2. **PI biosketch Personal Statement**：30 秒判断 PI 是否"撑得起这个 R01"；
3. **Approach 章节的 Figure 1-2**：30 秒判断 preliminary data 强度。

**写作启示**：Specific Aims 1 页 + Biosketch Personal Statement 是 "money pages"——必须独立请人在 90 秒内读完后**复述 central hypothesis**，复述不出 = 重写。

## 二、NSF Merit Review 双轨制（Intellectual Merit + Broader Impacts）

### 2.1 双轨必须双双过线

NSF 制度强制要求每份本子按 **Intellectual Merit (IM)** 和 **Broader Impacts (BI)** **分开评分**：

- **IM**：科学创新性、PI 资历、方法严谨性、resources 充分性；
- **BI**：教育 / 多样性 / 公众参与 / 工业转化 / 政策影响等。

**任何一轨低于 "M (Medium)"——本子立即出局**，哪怕另一轨是 HC (Highly Competitive)。

### 2.2 NSF 评分 5 等级

| 等级 | 缩写 | 含义 |
| --- | --- | --- |
| Highly Competitive | **HC** | 强烈推荐 |
| Competitive | **H** (or HQ) | 可以资助 |
| Some Competitiveness | **M** | 边缘（很少被资助） |
| Low Competitiveness | **L** | 几乎不可能被资助 |
| Not Competitive | **NC** | **死亡判决** |

NC 在很多 directorate 等于"程序性出局"——意味着 PI 接下来 1-2 年内重投同一 program 几乎不会被认真考虑。

### 2.3 BI 是"软指标 + PO 自由裁量权大"

NSF program officer (PO) 在最终资助决策中权重远大于 NIH——他们看 panel summary 后**可以挑选 panel 中 H/HC 评分的本子按 program 战略需要重新排序**。

**写作启示**：
- BI 不能写成"我们会发表论文 + 给学生培训"——这是套话；
- 必须具体到**人群（如 underrepresented minorities, K-12 students in rural Appalachia）+ 渠道（specific outreach partner）+ 评估指标（pre/post survey, retention rate）**；
- 投本子前**联系目标 program 的 PO**（NSF 鼓励），让他记住你 → 进入"PO 自由裁量"的优势区。

## 三、ERC Panel Chair 的 Hook 机制

### 3.1 ERC 的三大 domain 与 25 panels

ERC 每年评审分到 25 个 panels，归 3 个 domain：

- **PE (Physical Sciences and Engineering)**：~10 panels (PE1-PE10)
- **LS (Life Sciences)**：~9 panels (LS1-LS9)
- **SH (Social Sciences and Humanities)**：~6 panels (SH1-SH6)

每个 panel ~12-15 名 panel members + 数十位 external reviewers。**Panel chair** 在最终排序中权重极大。

### 3.2 Panel Chair 的 30 秒摘要机制

ERC Step 1 (Synopsis + Extended Synopsis + CV) 阶段：

- Panel chair 在 panel 会议上**口头介绍**每份本子 30 秒——这 30 秒影响其他 panel members 的初始观感；
- 因此 PI 必须把 **Synopsis 第 1 段**（5 句）写得"让 panel chair 30 秒能讲清楚"——本子的 selling point 必须**简单到 chair 能用 1 句话总结**。

### 3.3 Step 1 vs Step 2 心理差异

- **Step 1 (~30% 通过率)**：看 **idea 的 ground-breaking 程度** + PI track record；本子读到 7 页（Extended Synopsis 5 页 + CV 2 页）后给 A/B/C 评级；
- **Step 2 (~50% 通过率)**：进入 step 2 后是 **interview (~30 min) + extended proposal (15 页)**。Interview 在 Brussels 现场或线上——**前 5 分钟 PI 演讲** + **25 分钟 panel Q&A**；
- **Step 2 失败的最大原因**：interview 表现差——PI 在 Q&A 中不能清楚描述 plan B / 风险管理 / 跨学科边界。

**写作启示**：Step 1 写 idea；Step 2 准备 interview——**两个完全不同的"评审心理战场"**。

## 四、MSCA PF (Postdoctoral Fellowship) 双向 Fit

### 4.1 评审看 fellow 与 supervisor "互相加成"

MSCA PF 不是单纯评 fellow 的潜力，**评的是 fellow + supervisor + host institution 三方匹配**：

- **Fellow 维度**：研究独立性 + 国际流动潜力 + career development plan；
- **Supervisor 维度**：lab resources + supervisor 在该领域的国际地位 + 是否给 fellow 真正独立空间；
- **双向加成**：fellow 给 lab 带来什么新方向？lab 给 fellow 带来什么 fellow 自己拿不到的资源？

### 4.2 评审常见扣分点

- Supervisor "牌子大但 lab 已经做这件事"——评审会判定"fellow 没有独立性增量"；
- Fellow background "和 lab 完全相同方向"——双向加成弱；
- Career development plan 写成"我会发 5 篇 paper"——空话，必须具体到培训课程 + soft skills + 长期 PI 路径。

**写作启示**：MSCA 的本子要让 evaluator **同时看到 fellow 与 supervisor 的"互补性"**——他们各自不能独立完成的研究，三方合作能完成。

## 五、国内"匿名活页"评审（NSSF / 教育部人文社科）

### 5.1 匿名活页的物理结构

国家社科基金 / 教育部人文社科申请书分两本：

- **活页**（匿名）：所有可识别 PI 身份的信息**必须隐藏**（姓名、单位、代表作、致谢、基金号等），用 □□□ 代替；
- **附页**（具名）：PI 履历、代表作、单位、合作者全部具名。

### 5.2 评审心理两阶段

- **第一阶段（活页阶段）**：评审在不知道 PI 是谁的前提下，**纯按选题 + 框架 + 方法 + 创新性**打分；
- **第二阶段（附页阶段，会评）**：评审看 PI 履历 + 代表作，对活页打分做 ±0.5 档微调。

### 5.3 写作 / 退稿 雷区

- **活页中泄漏身份** = 程序性出局（"我们前期成果发表于《XX》[2023]"，结合学科范围一查就知道是谁——直接被毙）；
- **附页中代表作与活页选题方向不符** = 评审在第二阶段判"PI 跨界"，活页打分会被下调；
- **活页过长 / 附页过短** = 程序违规。

**写作启示**：活页 / 附页要**互锁但不重复**——活页讲科学问题 + 方案；附页讲 PI 凭什么能做。

## 六、综合"评审心理"自检清单

不论投哪个机构，写完后请按目标机构对应的清单自检：

| 机构 | 30 秒决策位 | 60 秒决策位 | 致命扣分点 |
| --- | --- | --- | --- |
| NIH R01 | Aims Para 1 | Aims 整页 + Bio Personal Statement | Aims 不具体 / preliminary data 弱 |
| NSF | Project Summary IM 段 | IM + BI 双轨各 1 页 | BI 写成套话 |
| ERC | Synopsis Para 1 | Extended Synopsis 5 页 | idea 不 ground-breaking |
| MSCA PF | Excellence Section 1.1 | Excellence + Implementation | 双向 fit 不明显 |
| NSSF | 活页选题 + 创新 1 页 | 活页全 5 页 | 活页泄漏身份 / 选题禁区 |

写完本子，请符合该机构母语 + 评审经历的同行做"模拟评审"——这是把"我觉得这本子很好"变成"评审觉得这本子很好"的唯一方式。
