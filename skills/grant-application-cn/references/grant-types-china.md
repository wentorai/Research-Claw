# 国内主流资助项目一览

> **免责声明**：本表反映公开制度的稳定特征。具体申请条件 / 资助强度 / 周期 / 命中率以**当年官方指南**为准，可能与下表数字略有出入。

## 1. 国家社科基金（全国哲学社会科学工作办公室）

主管：全国哲学社会科学工作办公室（简称"全国社科工办"），网站 npopss-cn.gov.cn。涵盖经济、管理、政治、法律、社会、教育、史学、文学、哲学等 23 个学科。

| 子类 | 申请条件 | 资助强度（参考） | 周期 | 备注 |
|------|---------|------------------|------|------|
| 一般项目 | 副高及以上 / 博士学位 | 20 万元 | 3 年 | 主流项目，竞争激烈 |
| 青年项目 | 35 周岁及以下 | 20 万元 | 3 年 | 青年学者首选 |
| 重点项目 | 正高级 / 学术带头人 | 35 万元 | 3-5 年 | 强调团队与积累 |
| 重大项目 | 国内一流学者 + 团队 | 80 万元 | 5 年 | 招标制，按选题指南投标 |
| 后期资助 | 已完成 70% 以上的成稿 | 8-30 万元 | 1-2 年 | 适合书稿即将完工的学者 |
| 西部项目 | 西部地区学者 / 西部主题 | 20 万元 | 3 年 | 命中率相对略高 |

**命中率**：一般 / 青年项目近年总体在 **10%-15%** 区间（请查阅当年指南公布数字）。

## 2. 教育部人文社会科学研究项目

主管：教育部社会科学司。仅限**教育部所属高校**及部分省属重点高校教师申请。

- **规划基金**：副高及以上，10 万元，3 年
- **青年基金**：40 周岁以下，8 万元，3 年
- **西部和边疆地区**：西部 + 边疆高校教师，10 万元
- **专项任务（含中国特色社会主义理论体系研究等）**：定向选题，强度按方向

申报通常在**每年 12 月-次年 1 月**，与社科基金错峰。

## 3. 省部级基金

### 自然科学类
- **北京市自然科学基金**（北京市科委 + 中关村基金会）：面上 / 重点 / 杰青 / 青年。强度 10-100 万元不等。
- **上海市科委**：自然科学基金 / 启明星计划 / 浦江人才。
- **广东省自然科学基金**：面上 / 杰青 / 青年提升 / 卓越青年。
- **江苏省自然科学基金**、**浙江省自然科学基金** 等。

### 社科类
- **省社科规划办**：各省（如上海哲社、广东社科、江苏社科）一般 / 青年 / 重大。
- **省教育厅人文社科项目**：面向省属高校。

> 省部级一般规则：申请条件比国家级宽松一档，资助强度低一档（通常 5-20 万元），周期 2-3 年；命中率视省份和学科差异较大（**5%-30%**），请查阅当年指南。

## 4. 博士后系统

主管：全国博士后管理委员会 + 中国博士后科学基金会。

| 项目 | 申请条件 | 资助强度 | 周期 |
|------|---------|---------|------|
| 博新计划（博士后创新人才支持计划） | 应届博士 / 入站半年内，A 类高校 | 60 万元（含生活补助 + 科研经费） | 2 年 |
| 博士后特别资助（站前 / 站中 / 站后） | 已入站或即将入站博士后 | 站前 18 万 / 站中 18 万 | 一次性 |
| 中国博士后科学基金面上 | 在站博士后 | 一等 12 万 / 二等 8 万 | 一次性 |
| 国（境）外交流项目（学术交流 / 引进项目 / 派出项目） | 在站博士后 / 拟出站博士 | 视方向 | 1-2 年 |

**命中率**：博新约 10%；面上约 30%（请查阅基金会官网最新公告）。

## 选项目决策树

```
是否在站博士后？
├─ 是 → 博新 / 博士后基金面上 / 特别资助
└─ 否
   ├─ 自然科学方向 → NSFC（见 nsfc-application-skill）/ 省自然科学基金
   └─ 人文社科方向
      ├─ 副高及以上 / 35+ → 国家社科一般 / 重点 / 教育部规划
      ├─ 35 岁以下博士 → 国家社科青年 / 教育部青年 / 省社科青年
      └─ 已完成 70% 书稿 → 国家社科后期资助
```

## 通用注意事项

- **成果归属与挂靠**：申请人必须为依托单位在编人员；离职 / 转岗会影响项目继续承担。
- **限项规定**：同一学者在研项目数有上限（一般国家级在研 ≤ 1 项，部分类别可叠加），违反将直接撤项。
- **意识形态合规**：人文社科类对选题禁区极为敏感，详见 `nssf-guide.md`。
