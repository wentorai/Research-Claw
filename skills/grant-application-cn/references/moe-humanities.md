# 教育部人文社会科学研究项目

主管：教育部社会科学司。是国家社科基金之外的**第二大人文社科资助渠道**，对**部属高校**和部分省属重点高校教师开放。

## 1. 项目分类

| 项目 | 申请条件 | 资助强度 | 周期 | 备注 |
|------|---------|---------|------|------|
| 规划基金（含规划基金、专项任务） | 副高及以上 / 博士学位 | 10 万 | 3 年 | 一般项目主流 |
| 青年基金 | 40 周岁及以下，讲师及以上或博士 | 8 万 | 3 年 | 青年学者首选 |
| 西部和边疆地区项目 | 西部 / 边疆地区高校教师 | 10 万 | 3 年 | 倾斜支持 |
| 后期资助 | 已完成成稿的副高 + | 视情况 | 1-2 年 | 适合书稿即将完工 |
| 中国特色社会主义理论体系研究专项 | 思政 / 马克思主义研究方向 | 视方向 | 3 年 | 定向选题 |
| 高校思想政治工作专项 | 思政教育方向 | 视方向 | 2 年 | 思政课与辅导员队伍建设 |

## 2. 与国家社科基金的核心差异

### 2.1 申请资格

- **国家社科**：所有具备基本科研条件的单位申请人
- **教育部**：**仅限教育部所属高校** + **教育部备案的省属重点高校**（清单每年公布）

非高校单位（社科院 / 党校 / 企业研究院）不能申报教育部项目。

### 2.2 评审与体例

- 国家社科**活页双盲**送审；教育部**完整申请书**评审，体例对申请人身份信息相对宽松（但仍要避免不当自我引用）。
- 国家社科文献综述要求"批判性 + 国际视野"；教育部要求略低，但**理论深度**仍是关键。

### 2.3 选题导向

- 国家社科：基础研究 + 重大现实问题，**学科属性**强
- 教育部：**教育领域**优先（教育学、思政、高等教育研究有明显加权）；**面向高等教育与立德树人**的应用研究受欢迎

### 2.4 成果要求

- 国家社科：阶段成果 + 最终成果，重点项目和重大项目要求专著
- 教育部：阶段成果（核心期刊论文）+ 最终成果（论文 / 报告 / 著作），相对灵活

## 3. 申请节奏

- **每年 12 月**：发布次年度申报通知 + 课题指南
- **次年 1-2 月**：高校内部初审 + 上报
- **3-5 月**：通讯评审 + 会评
- **6-7 月**：公布立项名单 + 下达经费

**与国家社科错峰**：可以"教育部 12 月申报 → 国家社科 2-3 月申报"两批轮投，但同年立项的两份不能完全同选题。

## 4. 体例要点

教育部申请书结构与国家社科相似，但通常含：

1. **数据表 / 基本信息表**
2. **课题论证**（最核心，长度 8000-15000 字典型）
   - 国内外研究现状述评
   - 选题意义与价值
   - 研究内容、思路与方法
   - 重点难点 / 主要观点 / 创新之处
3. **完成课题的可行性分析**
   - 已有研究基础
   - 主要参加者简介
   - 完成课题的保障条件
4. **预期成果**
5. **经费概算**
6. **承诺与签章**

## 5. 经费用法

教育部项目经费**直接费用为主**，间接费用占比较低（具体按当年财务通知）。可列支：

- 资料 / 数据费
- 调研差旅
- 会议费
- 劳务费（不能给在编人员）
- 出版 / 印刷
- 咨询专家费

不可列支：设备费比例不超过约 20%；不可购买高值固定资产。

## 6. 选题策略（教育部偏好）

教育部立项导向（公开指南反复强调）：

1. **服务高等教育内涵式发展**：人才培养、学科建设、学位与研究生教育
2. **服务立德树人 / 思想政治工作**：思政课改革、课程思政、辅导员队伍
3. **服务国家战略**：但聚焦教育与文化领域的子问题
4. **跨学科 / 新兴交叉**：教育数字化、教育人工智能、教育评价改革等热点
5. **基础学科振兴**：基础文科（哲学、历史、文学、考古）方向有定向支持

## 7. 高频退稿原因

1. 选题与教育领域无关（如纯经济学 / 纯法理 / 纯文学批评）— 转投国家社科或国自科
2. 课题论证篇幅不足，"东拉西扯"
3. 国内外研究综述只引中文文献，未对国际前沿做扎实评述
4. 研究方法只写"文献研究 + 比较研究 + 案例研究"列表，不写具体步骤
5. 预期成果只写"发表论文 X 篇"，不写**对教育实践 / 政策的应用价值**
6. 主要参加者过少（< 3 人）或过多（> 8 人）
7. 经费明显不合理（如某项占总额 70%）
