# 国家社科基金深度指引

国家社科基金（全国哲学社会科学工作办公室主管）是中国人文社会科学领域**最高级别**的资助项目。本文聚焦"一般 / 青年 / 重点 / 后期资助"四类的写作要点；重大项目走"招标制"，体例略不同，最后单列。

## 1. 申请书核心结构

国家社科基金《申请书》核心论证部分（即"活页"）由以下部分组成：

1. **选题依据**（占比最大）
   - 国内外研究现状述评
   - 选题价值（理论价值 + 应用价值）
   - 主要参考文献
2. **研究内容**
   - 研究对象 / 框架
   - 主要目标 / 研究思路 / 重点难点 / 主要观点 / 创新之处
3. **思路方法**
   - 基本思路 / 具体研究方法 / 重点突破之处
4. **预期成果**
   - 形式（专著 / 系列论文 / 研究报告）+ 字数 + 使用去向
5. **研究基础**
   - 课题负责人主要研究专长 / 学术简历 / 已发表成果（限近 5 年）
   - 研究条件（数据 / 资料 / 平台）

**活页隐去申请人信息**送审，所以**正文严禁出现可识别身份的信息**（含单位、姓名、连续自引等）。

## 2. 选题禁区与红线

国家社科基金对选题的**意识形态合规性**要求高。常见**红线**包括：

- 否定党的领导 / 否定改革开放 / 否定"四个自信"等政治立场不正确的选题
- 不当对比中国与境外政治制度优劣（除非是规范的比较政治学讨论）
- 涉港澳台、民族、宗教、历史虚无主义敏感选题（需特别审慎处理）
- 以咨询报告 / 政策建议为主要内容但缺乏理论深度

**软禁区**（不会被一票否决但常被压低评分）：

- 选题过于"政策跟风"，缺乏学术问题
- 选题过窄（如只研究某一企业 / 某一县）但又不能扩展为一般理论
- 选题过宽（如"中国式现代化研究"）但缺乏具体抓手

## 3. 研究方法（学科分组）

国家社科基金跨学科广，常用研究方法可归为 4 大类：

### 3.1 理论分析 / 思辨研究（哲学、文学、史学、法理学）

- **方法**：文本分析、概念辨析、思想史溯源、逻辑论证
- **写法要点**：研究方法部分写"将运用文本细读法、思想史研究法对 XX 进行分析"已不够，**必须具体到分析的步骤、文本范围、对话对象**
- **常见软肋**：方法部分写得像导论；缺乏与方法论权威著作（如施特劳斯、伽达默尔、福柯等）的对话

### 3.2 实证研究（经济、管理、社会、政治、心理）

- **方法**：调查研究、计量分析、实验法、社会网络分析、文本挖掘等
- **写法要点**：要写清**数据来源**（CGSS / CFPS / CHARLS / CHIP / CHNS / CLDS / 自建调查 / 行政数据等）、**变量构造**、**识别策略**（如 IV / DiD / RDD，建议参考 `causal-inference-cn` Skill）、**稳健性检验**
- **常见软肋**：识别策略写成"将运用回归分析"；遗漏对内生性的处理

### 3.3 调研研究 / 案例研究（社会、管理、政治、教育）

- **方法**：田野调查、深度访谈、参与式观察、案例比较、扎根理论
- **写法要点**：写清**调研对象选择标准**（理论抽样还是统计抽样）、**样本规模**、**访谈提纲设计**、**资料分析框架**（编码层级、扎根理论步骤）
- **常见软肋**：把"访谈 50 人"当做方法；缺乏案例选择的方法论辩护

### 3.4 历史研究 / 文献研究（史学、考古、文献学、民族学）

- **方法**：史料考辨、版本学、目录学、年代学、二重证据法、口述史
- **写法要点**：列出**核心史料范围**（已知藏地 + 拟新发掘）、**校勘 / 整理方案**、**与既有研究的对话**
- **常见软肋**：史料目录不清；新发现史料的"独占性"未充分论证

## 4. 重大项目特别说明

重大项目由**全国社科工办公布选题指南** → 学者按指南投标。**首席专家**通常为正高级 + 学科带头人。要点：

- 必须**完整覆盖**指南所列子课题，不可"挑食"
- 团队要包含校外专家（避免单一单位包揽）
- 经费 80 万元（标准额度），最终成果**必须含一部专著**
- 中期检查不通过将撤项

## 5. 申报时间节点

- **每年 1 月**：发布当年课题指南
- **2-3 月**：申报截止
- **4-6 月**：通讯评审（双盲，每份 5 位评审专家）
- **7-8 月**：会评（学科组）
- **9 月**：公布立项名单

## 6. 常见高频退稿原因

1. **选题政治不合规**或敏感（一票否决）
2. **选题平庸**：跟风热点但无创新观点
3. **文献综述不深**：堆砌国内文献，对国际前沿失语；或只批不立
4. **研究方法粗糙**：定性 + 定量"双拼"但都没做扎实
5. **预期成果泛化**：只写"形成一部专著"无字数 / 章节 / 用途
6. **代表作与课题不相关**：申请人换赛道但未铺垫
7. **活页里"漏出身份"**：写出"我们课题组已在 XX 期刊发表..."等泄露身份的句子
8. **重复立项嫌疑**：与已结题项目主题过近

## 7. 与教育部人文社科基金的取舍

| 维度 | 国家社科 | 教育部人文社科 |
|------|---------|----------------|
| 主管 | 全国社科工办 | 教育部社科司 |
| 资助强度 | 20-80 万 | 8-10 万 |
| 命中率 | 较低（10-15%） | 较高（约 20%） |
| 申请节奏 | 1-3 月 | 12 月-次年 1 月 |
| 体例 | "活页"双盲 | 完整申请书评审 |
| 适合人群 | 已发顶刊 / 副高及以上 | 青年讲师 / 副高起步阶段 |

**策略建议**：若选题成熟，优先国家社科；若选题尚在打磨期 / 申请人资历较浅，先冲教育部一年练手再上社科基金。
