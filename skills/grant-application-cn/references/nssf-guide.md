# 国家社科基金深度指引

国家社科基金（全国哲学社会科学工作办公室主管，简称"全国社科工办"）是中国人文社会科学领域**最高级别**的资助项目。本文聚焦"一般 / 青年 / 重点 / 后期资助"四类的写作要点；重大项目走"招标制"，体例略不同，最后单列。

## 1. 申请书核心结构

国家社科基金《申请书》核心论证部分（即"活页"）由以下部分组成：

1. **选题依据**（占比最大，约整个论证 1/3）
   - 国内外研究现状述评
   - 选题价值（理论价值 + 应用价值）
   - 主要参考文献
2. **研究内容**
   - 研究对象 / 框架
   - 主要目标 / 研究思路 / 重点难点 / 主要观点 / 创新之处
3. **思路方法**
   - 基本思路 / 具体研究方法 / 重点突破之处
4. **预期成果**
   - 形式（专著 / 系列论文 / 研究报告）+ 字数 + 使用去向
5. **研究基础**
   - 课题负责人主要研究专长 / 学术简历 / 已发表成果（限近 5 年）
   - 研究条件（数据 / 资料 / 平台）

**活页隐去申请人信息**送审，所以**正文严禁出现可识别身份的信息**（含单位、姓名、连续自引等）。

### 1.1 活页排版规范（以当年指南为准，下为典型规范）

- **A4 大小 8 个版面**，A3 纸打印（pdf 上传），中缝装订
- 不要更改《申请书》系统提供的字体（**华文中宋**为底；缺字会出现"参差"）
- 关键句 / 主旨句可加粗或下划线，但**不可大段加粗**
- 不可插入显示申请人信息的水印 / 页眉

## 2. 选题禁区与红线（基于公开指南反复强调）

国家社科基金对选题的**意识形态合规性**要求高（通讯评审 + 会评中均有"政治审查"环节）。常见**红线**包括：

- **否定党的领导 / 否定改革开放 / 否定"四个自信" / 历史虚无主义**等政治立场不正确的选题（一票否决）
- **不当对比**中国与境外政治制度优劣（除非是规范的比较政治学讨论且立场正确）
- **涉港澳台、民族、宗教**敏感选题——可以做，但需特别审慎处理（避免把政治性问题作为研究框架的主轴）
- **以咨询报告 / 政策建议为主要内容**但缺乏理论深度（被视为"申报对象错位"，应申报"国家高端智库"等政策研究渠道）
- **重复立项嫌疑**：与申请人已结题项目主题过近，或当年指南明示"已立项不再资助"的方向

**软禁区**（不会被一票否决但常被压低评分）：

- 选题过于"政策跟风"，缺乏学术问题的发掘
- 选题过窄（如只研究某一企业 / 某一县）但又不能扩展为一般理论
- 选题过宽（如"中国式现代化研究"）但缺乏具体抓手
- **跨学科选题**但申请人代表作集中在单一学科（评审会怀疑能力）
- **运用非主流理论框架**（如完全用西方某学派批判中国实践）但缺乏正面建构
- **题目里直接出现"问题"二字**（如"XX 的问题与对策"）—— 评审视为口径偏负面，建议改为"XX 的机制与路径"

### 2.1 当年课题指南的"信号词"

每年 1 月发布的**课题指南**是选题的最佳指引。注意识别以下信号词：

- **"研究阐释..."**（如"研究阐释党的二十大精神""研究阐释中国式现代化"）→ 紧贴官方文件，做正面建构
- **"中国自主知识体系"**（近年高频）→ 强调中国特色 + 自主理论
- **"高质量发展" / "新质生产力" / "全过程人民民主"** → 政策概念学理化
- **方法论方向**（如"哲学社会科学评价体系研究"）→ 元学术研究
- **基础学科振兴**（哲学、历史、考古、古文字、文献学）→ 定向支持

## 3. 研究方法（学科分组 4 类详解）

国家社科基金跨学科广，常用研究方法可归为 4 大类。**评审专家最常诟病"研究方法"段空洞**——不要只列方法名，要写具体步骤、范围、对话对象。

### 3.1 理论分析 / 思辨研究（哲学、文学、史学、法理学、政治哲学）

**适用学科**：哲学、文学批评、法理学、政治哲学、伦理学、史学方法论。

**典型方法**：
- **文本细读**（close reading）：基于一手文本逐句分析
- **概念辨析 / 概念史**（Begriffsgeschichte 方法）：对核心概念在历史脉络中的语义变迁
- **思想史溯源**：把研究对象置于学术史长河
- **逻辑论证 / 形式化**：构建论证 / 反例 / 推论链条
- **比较哲学 / 比较法学**：跨文化 / 跨法系对照

**研究方法段范式（强写法）**：
> 本研究采用文本细读、概念史考察与逻辑论证相结合的方法。具体而言：(1) 对 [核心一手文献 X] 进行逐章细读，重点考察 [概念 A] 在 [年份范围] 间的语义变迁；(2) 与 [代表性流派 / 思想家 B、C、D] 的论述进行对话，识别共识与分歧；(3) 在 [问题域 Y] 中构建本研究的论证链：[前提] → [推论] → [结论]，并预设 3 类反例进行反思。

**常见软肋**：
- 方法部分写得像导论，没有"步骤"
- 缺乏与方法论权威著作（如施特劳斯、伽达默尔、福柯、Quentin Skinner、Reinhart Koselleck）的对话
- 把"运用辩证法 / 历史唯物主义"当方法（缺乏可操作步骤）

### 3.2 实证研究（经济、管理、社会、政治、心理、传播）

**适用学科**：经济学、管理学、社会学、政治学、心理学、传播学、教育实证。

**典型方法**：
- **调查研究**：问卷调查（CGSS / CFPS / CHARLS / CHIP / CHNS / CLDS / CGSS / 自建调查 / 行政数据）
- **计量分析**：OLS / Logit / Probit / DiD / RDD / IV / Synthetic Control / Panel Fixed Effects
- **实验法**：Lab experiment / Field experiment / RCT / Natural experiment
- **社会网络分析 (SNA)**
- **文本挖掘 / 大数据**：情感分析、主题模型、深度学习文本分类

**研究方法段范式（强写法）**：
> 本研究将利用 [数据 X，明确来源 + 样本框 + 时间范围 + 样本量] 进行实证分析。具体设计：(1) 构造 [因变量] 与 [核心自变量] 的测度，控制 [个体 / 时间] 双向固定效应；(2) 鉴于 [内生性来源]，采用 [识别策略 — 如基于 [外生冲击 X] 的 DiD]，并提供 [4 类稳健性检验：替换样本 / 替换变量定义 / 安慰剂检验 / [机制 channel] 异质性分析]；(3) 样本量计算: 假设 effect size = [d]、α = 0.05、1-β = 0.8，需样本 N ≥ [n]，本研究实际样本 [N实际] 满足要求。

**常见软肋**：
- 识别策略写成"将运用回归分析" → 没说处理内生性
- 数据来源不明 / 变量构造不清晰
- "稳健性检验"只罗列名词不说具体操作
- 缺少 power analysis（样本量论证）
- DiD 没说 parallel trends pre-test；IV 没论证 exclusion restriction

> **可衔接 `causal-inference-cn` Skill** 获得更细的识别策略写法。

### 3.3 调研研究 / 案例研究（社会、管理、政治、教育、人类学）

**适用学科**：社会学、人类学、管理学、政治学、教育学、公共管理。

**典型方法**：
- **田野调查 / 民族志**（ethnography）：长时段沉浸式观察 + 厚描
- **深度访谈**（in-depth interview）+ 编码（开放编码 / 主轴编码 / 选择编码 / 扎根理论 GT）
- **参与式观察**（participant observation）
- **比较案例研究**（comparative case study, MSSD / MDSD design）
- **过程追踪**（process tracing，政治学常用）

**研究方法段范式（强写法）**：
> 本研究采用扎根理论指导下的多案例比较研究法。(1) **案例选择**: 基于 "**最大差异系统设计**" (MDSD)，选择 4 个在 [关键维度 X] 上差异最大的案例 [案例 A/B/C/D]，理论饱和度评估见附录。(2) **数据收集**: 每个案例进行 ≥ 25 次深度访谈（涵盖 [角色 1/2/3]）+ 6 个月田野观察 + 公开文件 / 内部文档收集。(3) **资料分析**: 三级编码（Strauss & Corbin 1998），由 2 名研究员独立编码后比对（Cohen's κ ≥ 0.7）。(4) **理论建构**: 使用因果机制图（Bennett & Checkel 2015 框架）展示跨案例的 X → Y → Z 路径。

**常见软肋**：
- 把"访谈 50 人"当做方法描述（缺乏案例选择的方法论辩护）
- 缺乏 inter-rater reliability 报告
- "扎根理论"只是装饰词（实际只是简单归纳）
- 没说 reflexivity（研究者立场对资料的影响）

### 3.4 历史研究 / 文献研究（史学、考古、文献学、民族学）

**适用学科**：中国史、世界史、考古学、文献学、目录学、民族学（历史方向）、宗教史。

**典型方法**：
- **史料考辨**（textual criticism）：版本对勘、辨伪、辑佚
- **版本学 / 目录学 / 校勘学**：传世文献的版本系统建立
- **年代学 / 历法考证**
- **二重证据法**（王国维）：传世文献 + 出土文献相互印证
- **三重证据法 / 四重证据法**：再加上民族学 / 比较语言学 / 图像
- **口述史 / 历史记忆研究**
- **数字人文方法**：GIS、prosopography 数据库、社会网络分析

**研究方法段范式（强写法）**：
> 本研究运用版本对勘、二重证据法与数字人文方法。(1) **史料范围**: 传世文献以 [典籍 X 的 4 个主要版本：宋刊本 / 明刊本 / 清刊本 / 民国排印本] 为底本，出土文献以 [近 10 年新出 X 类简帛] 为补充，两者目录见附录。(2) **校勘方案**: 采用"以宋刊为底，参校 3 家"的体例（仿 [中华书局某经典点校本]），异文出注。(3) **数据库构建**: 将 [对象 X] 的关键属性录入 prosopography 式数据库（采用 China Biographical Database, CBDB 数据格式标准），共 [N 条记录]。(4) **与既有研究对话**: 重点回应 [史家 A / B / C] 的论述。

**常见软肋**：
- 史料目录不清；新发现史料的"独占性"未充分论证
- "拟搜集 X 史料"但没说渠道 / 已有联系
- 数字人文项目没说数据格式 / 长期保存
- 缺乏对应的基础设施（古籍数据库订阅 / 专家咨询）

## 4. 重大项目特别说明

重大项目由**全国社科工办公布选题指南** → 学者按指南投标。**首席专家**通常为正高级 + 学科带头人。

要点：

- 必须**完整覆盖**指南所列子课题，不可"挑食"
- 团队要包含校外专家（避免单一单位包揽）
- 经费 60-80 万元（视年度调整），最终成果**必须含一部专著**
- 中期检查不通过将撤项
- **每个招标选题原则上确立 1-2 项中标课题**——竞争极激烈

## 5. 申报时间节点（典型年份，以当年公告为准）

- **每年 1 月**：发布当年课题指南
- **2-3 月**：申报截止（学校内部初审 + 上报）
- **4-6 月**：通讯评审（双盲，每份典型 5 位评审专家）
- **7-8 月**：会评（学科组）
- **9 月**：公布立项名单
- **重大项目**：另行公告，通常 5-6 月发布招标公告，截止日期较年度项目晚

## 6. "获奖申请书"共有的 5 个特征（基于公开优秀活页观察）

获优 / 立项的申请书与被退的最大差别**不是论题大小或方法新颖**，而是以下 5 个写作"成熟度"特征。这是从公开发表的优秀活页范本（如各高校官网公开样本）+ 评审专家访谈（如长沙理工大学整理的 14 位评审专家深度访谈材料）总结的共性：

### 特征 1: 文献综述含"综 / 梳 / 述 / 评"四要素

**弱**：连续 10 句"[Smith 2020] 发现 X、[Lee 2021] 发现 Y..." → 这只是"列"

**强**：
- **综**：宏观把握领域全貌（用 1 段 + 1 张图）
- **梳**：把已有研究分流派 / 分阶段（用清晰的逻辑切片）
- **述**：每个流派的核心主张 + 代表作 + 代表方法
- **评**：每个流派的局限 + 与本研究的关系（最关键也最容易缺）

文献综述的"点睛之笔"是**对当前研究不足的评述** —— 这些不足正是本项目要填补的空白。

### 特征 2: 文献时效近、视野宽

- 重点引近 **5 年**乃至近 **3 年**文献——评审一眼能看出"作者是否站在潮头"
- **国际文献占比 ≥ 50%**（人文学科可适当放宽至 30%）
- 引文标准化：作者. 题目. 期刊, 年, 卷(期): 页[DOI]
- **避免连续 5 篇都是同一国内学者引用**（暴露身份风险 + 视野狭窄印象）

### 特征 3: 研究内容与研究方法严格对应

许多本子写"研究内容"时列 4 个子课题，但"研究方法"只笼统说"采用文献研究 + 实证分析" —— 评审专家在比对时找不到"方法 → 子课题"的映射。

强写法：建立 **子课题 ↔ 方法 ↔ 数据 ↔ 预期产出** 的 4 列对照表，要么显式画出，要么在文字中清晰交代。

### 特征 4: "选题价值"段含"理论 + 应用"双线 + 量化承诺

弱：仅写"具有重大理论价值和应用价值"。

强：
- **理论价值**：明确本研究对 [具体理论框架 X] 的推进点（不是"丰富了 X 理论"——"丰富"不是论断）
- **应用价值**：具体到为 [具体政策 / 实践问题 Y] 提供 [具体支持]
- **量化承诺**：预期成果中明示"完成专著 1 部 (≥ 30 万字) + 系列论文 5 篇 (CSSCI 来源期刊 ≥ 3 篇) + 研究报告 1 份 (报送 [具体决策部门])"

### 特征 5: 主旨句优先 + 关键词强调式排版

评审专家阅读节奏快（一份本子最长 30 分钟）：
- **每段第一句是主旨句**（topic sentence first）—— 即使评审只读首句也能 follow
- 关键概念 / 关键句**加粗**或**下划线**（但每段最多 1 处，不要全粗）
- 长段落每 4-5 行就分一段，避免出现 ≥ 10 行的"段落墙"
- **目录式标题**逐级清晰（一、(一)、1. 等中文层级）

## 7. 与教育部人文社科基金的真实差异（不是说法上的）

| 维度 | 国家社科 | 教育部人文社科 |
|------|---------|----------------|
| **级别** | 国家级 | 省部级 |
| **主管** | 全国社科工办 | 教育部社科司 |
| **资助强度** | 一般/青年 20 万；重点 35 万；重大 60-80 万 | 规划 10 万；青年 8 万 |
| **命中率** | 较低（10-15%，部分热门学科 < 8%） | 较高（约 18-22%） |
| **申请节奏** | 1-3 月 | 12 月 - 次年 1 月 |
| **体例** | "活页"双盲送审 | 完整申请书评审，身份信息相对宽松 |
| **课题指南** | **有**——年度指南细分到子题；选题须呼应指南 | **无指南**——选题自由度高 |
| **限项** | 限项申报（每个学校 / 单位有限额） | **不限项** |
| **青年项目年龄** | ≤ 35 周岁 | ≤ 40 周岁 |
| **申报资格** | 所有具备基本科研条件的单位 | **仅限教育部所属高校 + 教育部备案的省属重点高校** |
| **立项导向** | 高频词集中："新时代""数字""乡村""中国式现代化"——宏观政策对接 | 高频词："机制研究""影响研究""教育"等——实践应用导向 |
| **成果要求** | 重点 + 重大必须专著；最终成果验收严 | 论文 / 报告 / 著作灵活 |

**策略建议**：
- 选题成熟、副高及以上 → 优先国家社科一般 / 重点
- 选题打磨期 / 35 岁以下 → 国家社科青年（一年一搏）
- 教育领域题材或 35-40 岁 → 教育部人文社科（既有竞争优势又有窗口期）
- 已有 70% 以上书稿 → 国家社科 / 教育部后期资助（命中率较高）
- **同年不可兼报国家社科 + 教育部同选题**——但"教育部 12 月申报 → 国家社科 2-3 月申报"两批轮投是常见做法（不同选题）

## 8. 常见高频退稿原因（评审专家访谈整合）

1. **选题政治不合规**或敏感（一票否决）
2. **选题平庸**：跟风热点但无创新观点
3. **文献综述不深**：堆砌国内文献，对国际前沿失语；或只批不立
4. **研究方法粗糙**：定性 + 定量"双拼"但都没做扎实
5. **预期成果泛化**：只写"形成一部专著"无字数 / 章节 / 用途
6. **代表作与课题不相关**：申请人换赛道但未铺垫（评审认为可行性低）
7. **活页里"漏出身份"**：写出"我们课题组已在 XX 期刊发表..." / 连续自引申请人著作
8. **重复立项嫌疑**：与已结题项目主题过近
9. **"研究内容"与"研究方法"两段无对应** → 评审找不到映射
10. **"研究基础"段过弱**：近 5 年代表作 < 3 篇 CSSCI 或与课题方向脱节

## 9. 自检 Checklist

- [ ] 选题与当年课题指南呼应（信号词命中至少 1 个）？
- [ ] 选题禁区已避开（政治合规 + 软禁区都查）？
- [ ] 文献综述含"综 / 梳 / 述 / 评"四要素？
- [ ] 国际文献占比 ≥ 50%（人文 ≥ 30%）+ 近 5 年文献占主体？
- [ ] 研究方法段含具体步骤、数据来源、对话对象（不是只列方法名）？
- [ ] 4 列对照表（子课题 ↔ 方法 ↔ 数据 ↔ 产出）能默写出来？
- [ ] 选题价值段含理论 + 应用双线 + 量化承诺？
- [ ] 预期成果含字数 + 篇数 + 期刊级别 + 报送渠道？
- [ ] 活页**完全匿名**（无单位 / 姓名 / 项目号 / 连续自引）？
- [ ] 代表作（近 5 年 ≤ 10 项）与课题方向**强相关**？
- [ ] 研究基础段含数据 / 资料 / 平台条件（不只是"有团队"）？
- [ ] 排版：每段首句是主旨句？关键句加粗适度（每段最多 1 处）？
